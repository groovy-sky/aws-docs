# Available Libraries

AWS provides libraries (the AWS SDKs), which include sample code, tutorials, and
other resources for software developers who prefer to build applications using
language-specific APIs instead of writing their own HTTP requests. These SDKs provide
basic functions (not included in the APIs), such as request authentication, request
retries, and error handling so that it is easier to get started. AWS SDKs are available
for the following languages:

- [Java](http://aws.amazon.com/java)

- [PHP](http://aws.amazon.com/php)

- [Python](http://aws.amazon.com/python)

- [Ruby](http://aws.amazon.com/ruby)

- [Windows and .NET](http://aws.amazon.com/net)

For links to the documentation for all AWS SDKs for supported languages, go to the Software Development Kits (SDKs) section on the [AWS Documentation](http://aws.amazon.com/documentation) page.

For libraries and sample code in all languages, go to [Sample Code & Libraries](http://aws.amazon.com/code).
