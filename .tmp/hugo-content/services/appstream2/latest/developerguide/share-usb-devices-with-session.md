# Enable Users to Share a USB Device with an WorkSpaces Applications Streaming Session

Before users share their USB devices with an WorkSpaces Applications session, the USB devices must be
qualified. Otherwise, when users start a streaming session, their USB device is not
detected by WorkSpaces Applications and cannot be shared with the session. For more information, see
[Qualify USB Devices for Use with Streaming Applications](qualify-usb-devices.md).

###### Note

Sharing a USB device with an WorkSpaces Applications streaming session is not supported on the
macOS client.
