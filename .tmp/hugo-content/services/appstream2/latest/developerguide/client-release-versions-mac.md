# WorkSpaces Applications macOS Client Release Notes

The following table describes the latest updates that are available in released
versions of the macOS WorkSpaces Applications client.

Client versionRelease dateChanges

1.3.0

02-10-2026

- Added support for user pools, an alternative method for access management that provides a persistent portal for users.

- Added streaming support for Ubuntu-based Elastic fleets.

- Improved the SAML login flow to support your preferred browser, including Apple Safari, Google Chrome, Mozilla Firefox, and Microsoft Edge browsers.

- Fixed an issue to sort applications in the catalog by application display name.

- Fixed an issue that was preventing a streaming session from being redirected from the web browser to the client if the client application was not previously open.

- Updated the DCV SDK.

- Other bug fixes and enhancements.

11-06-2025

- Launching the new Amazon WorkSpaces Applications client
