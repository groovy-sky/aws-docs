# How to Enable File System Redirection

Perform the following steps to enable both file upload and download on the
stack that your users access for streaming sessions.

1. Open the WorkSpaces Applications console at
    [https://console.aws.amazon.com/appstream2/home](https://console.aws.amazon.com/appstream2/home).

2. In the left navigation pane, choose
    **Stacks**.

3. Choose the stack for which you want to enable file system
    redirection.

4. Choose the **User Settings** tab, and then expand the
    **Clipboard, file transfer, and local print**
**permissions** section.

5. For **File transfer**, verify that **Upload**
**and download** is selected. If not, choose
    **Edit**, and then choose **Upload and**
**download**.

6. Choose **Update**.
