# ObjectLockRetention

A Retention configuration for an object.

## Contents

**Mode**

Indicates the Retention mode for the specified object.

Type: String

Valid Values: `GOVERNANCE | COMPLIANCE`

Required: No

**RetainUntilDate**

The date on which this Object Lock Retention will expire.

Type: Timestamp

Required: No
