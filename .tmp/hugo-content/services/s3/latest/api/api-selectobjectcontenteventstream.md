# SelectObjectContentEventStream

The container for selecting objects from a content event stream.

The Continuation Event.

Type: [ContinuationEvent](api-continuationevent.md) data type

Required: No

**End**

The End Event.

Type: [EndEvent](api-endevent.md) data type

Required: No

**Progress**

The Progress Event.

Type: [ProgressEvent](api-progressevent.md) data type

Required: No

**Records**

The Records Event.

Type: [RecordsEvent](api-recordsevent.md) data type

Required: No

**Stats**

The Stats Event.

Type: [StatsEvent](api-statsevent.md) data type

Required: No
