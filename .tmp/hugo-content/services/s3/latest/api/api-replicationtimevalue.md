# ReplicationTimeValue

A container specifying the time value for S3 Replication Time Control (S3 RTC) and replication metrics
`EventThreshold`.

## Contents

**Minutes**

Contains an integer specifying time in minutes.

Valid value: 15

Type: Integer

Required: No
