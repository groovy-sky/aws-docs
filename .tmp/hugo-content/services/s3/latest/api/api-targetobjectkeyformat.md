# TargetObjectKeyFormat

Amazon S3 key format for log objects. Only one format, PartitionedPrefix or SimplePrefix, is
allowed.

## Contents

**PartitionedPrefix**

Partitioned S3 key for log objects.

Type: [PartitionedPrefix](api-partitionedprefix.md) data type

Required: No

**SimplePrefix**

To use the simple format for S3 keys for log objects. To specify SimplePrefix format, set
SimplePrefix to {}.

Type: [SimplePrefix](api-simpleprefix.md) data type

Required: No
