# MatchObjectSize

A filter condition that specifies the object size range of included objects in bytes. Only integers are supported.

## Contents

**BytesGreaterThan**

Specifies the minimum object size in Bytes. The value must be a positive number, greater than
0 and less than 50 TB.

Type: Long

Required: No

**BytesLessThan**

Specifies the maximum object size in Bytes. The value must be a positive number, greater than
the minimum object size and less than 50 TB.

Type: Long

Required: No
