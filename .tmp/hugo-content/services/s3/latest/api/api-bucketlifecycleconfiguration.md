# BucketLifecycleConfiguration

Specifies the lifecycle configuration for objects in an Amazon S3 bucket. For more information, see
[Object Lifecycle\
Management](../dev/object-lifecycle-mgmt.md) in the _Amazon S3 User Guide_.

## Contents

**Rules**

A lifecycle rule for individual objects in an Amazon S3 bucket.

Type: Array of [LifecycleRule](api-lifecyclerule.md) data types

Required: Yes
