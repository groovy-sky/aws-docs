# AsyncResponseDetails

A container for the response details that are returned when querying about an
asynchronous request.

## Contents

**ErrorDetails**

Error details for an asynchronous request.

Type: [AsyncErrorDetails](api-control-asyncerrordetails.md) data type

Required: No

**MultiRegionAccessPointDetails**

The details for the Multi-Region Access Point.

Type: [MultiRegionAccessPointsAsyncResponse](api-control-multiregionaccesspointsasyncresponse.md) data type

Required: No
