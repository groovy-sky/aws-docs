# RequestProgress

Container for specifying if periodic `QueryProgress` messages should be sent.

## Contents

**Enabled**

Specifies whether periodic QueryProgress frames should be sent. Valid values: TRUE, FALSE. Default
value: FALSE.

Type: Boolean

Required: No
