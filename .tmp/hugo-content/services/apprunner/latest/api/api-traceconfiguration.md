# TraceConfiguration

Describes the configuration of the tracing feature within an AWS App Runner observability configuration.

## Contents

**Vendor**

The implementation provider chosen for tracing App Runner services.

Type: String

Valid Values: `AWSXRAY`

Required: Yes
