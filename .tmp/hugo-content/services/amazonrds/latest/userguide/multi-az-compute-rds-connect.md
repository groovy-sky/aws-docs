# Automatically connecting an AWS compute resource and a Multi-AZ DB cluster for Amazon RDS

You can automatically connect a Multi-AZ DB cluster and AWS compute resources such as Amazon Elastic Compute Cloud (Amazon EC2) instances and
AWS Lambda functions.

The following topics provide detailed instructions for configuring network settings,
security groups, and connection parameters to establish reliable connections to Amazon RDS DB
instances within a Multi-AZ DB cluster deployment. They focus on optimizing network connectivity and
performance for applications interacting with Multi-AZ DB cluster, ensuring secure and efficient data
operations.
