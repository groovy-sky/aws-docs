# Connecting to your Oracle DB instance

After Amazon RDS provisions your Oracle DB instance, you can use any standard SQL client application
to log in to your DB instance. Because RDS is a managed service, you can't log in as SYS or
SYSTEM. For more information, see [RDS for Oracle users and privileges](oracle-concepts-privileges.md).

In this topic, you learn how to use Oracle SQL Developer or SQL\*Plus to connect to an
RDS for Oracle DB instance. For an example that walks you through the process of creating and connecting
to a sample DB instance, see [Creating and connecting to an Oracle DB instance](chap-gettingstarted-creatingconnecting-oracle.md).
