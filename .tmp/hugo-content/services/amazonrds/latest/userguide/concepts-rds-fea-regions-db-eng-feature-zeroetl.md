# Supported Regions and DB engines for Amazon RDS zero-ETL integrations

RDS zero-ETL integrations is a fully managed solution for making transactional data
available in Amazon Redshift or Amazon SageMaker after it's written to an Amazon RDS DB instance. For more information, see [Amazon RDS zero-ETL integrations](zero-etl.md).
