# Securing Oracle DB instance connections

Amazon RDS for Oracle supports SSL/TLS encrypted connections and also the Oracle Native Network Encryption (NNE) option
to encrypt connections between your application and your Oracle DB instance. For more information about the
Oracle Native Network Encryption option, see [Oracle native network encryption](appendix-oracle-options-networkencryption.md).
