# Working with CDBs in RDS for Oracle

In the Oracle multitenant architecture, a container database (CDB) can include
customer-created pluggable databases (PDBs). For more information about CDBs, see [Introduction to the Multitenant Architecture](https://docs.oracle.com/en/database/oracle/oracle-database/19/multi/introduction-to-the-multitenant-architecture.html) in the Oracle Database
documentation.
