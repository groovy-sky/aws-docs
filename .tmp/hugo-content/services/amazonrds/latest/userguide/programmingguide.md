# Amazon RDS API reference

In addition to the AWS Management Console and the AWS Command Line Interface (AWS CLI), Amazon RDS also provides an API. You
can use the API to automate tasks for managing your DB instances and other objects in Amazon RDS.

- For an alphabetical list of API operations, see [Actions](../../../../reference/amazonrds/latest/apireference/api-operations.md).

- For an alphabetical list of data types, see [Data types](../../../../reference/amazonrds/latest/apireference/api-types.md).

- For a list of common query parameters, see [Common parameters](../../../../reference/amazonrds/latest/apireference/commonparameters.md).

- For descriptions of the error codes, see [Common errors](../../../../reference/amazonrds/latest/apireference/commonerrors.md).

For more information about the AWS CLI, see [AWS Command Line Interface reference for Amazon RDS](../../../cli/latest/reference/rds/index.md).
