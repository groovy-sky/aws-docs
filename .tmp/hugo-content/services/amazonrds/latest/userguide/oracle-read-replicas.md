# Working with read replicas for Amazon RDS for Oracle

To configure replication between Oracle DB instances, you can create replica databases. For an overview of Amazon RDS
read replicas, see [Overview of Amazon RDS read replicas](user-readrepl.md#USER_ReadRepl.Overview). For a summary of
the differences between Oracle replicas and other DB engines, see [Differences between read replicas for DB engines](user-readrepl-overview-differences.md).
