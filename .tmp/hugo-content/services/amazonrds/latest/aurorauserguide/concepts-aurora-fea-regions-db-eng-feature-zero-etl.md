# Supported Regions and Aurora DB engines for zero-ETL integrations

Amazon Aurora zero-ETL integrations is a fully managed solution for making
transactional data available in Amazon Redshift or Amazon SageMaker after it's written to an Aurora cluster. For more
information, see [Aurora zero-ETL integrations](zero-etl.md).
