# Amazon Aurora on the AWS Free Tier

Aurora PostgreSQL is also available on the [AWS\
Free Tier](https://aws.amazon.com/rds/free) through the [Create with express configuration](chap-gettingstartedaurora-aurorapostgresql-expressconfig.md). In addition to
the express configuration limitations, the following restrictions apply when using
Aurora PostgreSQL with the AWS Free Tier:

- Only clusters created with express configuration are supported (full configuration is
not available)

- Up to 4 Aurora Capacity Units (ACUs) per cluster

- Up to 1 GB storage per cluster

- Maximum 2 clusters and 2 instances per account

- Aurora Global Database and Zero-ETL integrations are not supported
