# Rename an undisplayed saved query

###### To rename a saved query not displayed in the query editor

1. In the Athena console, choose the **Saved queries**
    tab.

2. Select the check box for the query that you want to rename.

3. Choose **Rename**.

4. In the **Rename query** dialog box, edit the query name and
    query description. You can use the expandable **Preview SQL**
**query** window to verify the contents of the query before you
    rename it.

5. Choose **Rename query**.

The renamed query appears in the **Saved queries**
    list.
