# AWS::DataZone::PolicyGrant ProjectGrantFilter

The project grant filter.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "DomainUnitFilter" : DomainUnitFilterForProject
}

```

### YAML

```yaml

  DomainUnitFilter:
    DomainUnitFilterForProject

```

## Properties

`DomainUnitFilter`

The domain unit filter of the project grant filter.

_Required_: Yes

_Type_: [DomainUnitFilterForProject](aws-properties-datazone-policygrant-domainunitfilterforproject.md)
