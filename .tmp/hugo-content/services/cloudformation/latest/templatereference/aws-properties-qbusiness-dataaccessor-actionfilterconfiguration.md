# AWS::QBusiness::DataAccessor ActionFilterConfiguration

Specifies filters to apply to an allowed action.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "DocumentAttributeFilter" : AttributeFilter
}

```

### YAML

```yaml

  DocumentAttributeFilter:
    AttributeFilter

```

## Properties

`DocumentAttributeFilter`

Enables filtering of responses based on document attributes or metadata fields.

_Required_: Yes

_Type_: [AttributeFilter](aws-properties-qbusiness-dataaccessor-attributefilter.md)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
