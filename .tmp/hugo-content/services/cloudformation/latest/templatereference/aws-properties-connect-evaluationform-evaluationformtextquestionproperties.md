# AWS::Connect::EvaluationForm EvaluationFormTextQuestionProperties

Information about properties for a text question in an evaluation form.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "Automation" : EvaluationFormTextQuestionAutomation
}

```

### YAML

```yaml

  Automation:
    EvaluationFormTextQuestionAutomation

```

## Properties

`Automation`

The automation properties of the text question.

_Required_: No

_Type_: [EvaluationFormTextQuestionAutomation](aws-properties-connect-evaluationform-evaluationformtextquestionautomation.md)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
