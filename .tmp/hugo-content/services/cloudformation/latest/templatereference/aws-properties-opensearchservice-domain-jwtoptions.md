# AWS::OpenSearchService::Domain JWTOptions

The `JWTOptions` property type specifies Property description not available. for an [AWS::OpenSearchService::Domain](aws-resource-opensearchservice-domain.md).

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "Enabled" : Boolean,
  "PublicKey" : String,
  "RolesKey" : String,
  "SubjectKey" : String
}

```

```yaml

  Enabled: Boolean
```
