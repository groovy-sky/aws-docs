# AWS::QuickSight::Template ImageInteractionOptions

The general image interactions setup for image publish options.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "ImageMenuOption" : ImageMenuOption
}

```

### YAML

```yaml

  ImageMenuOption:
    ImageMenuOption

```

## Properties

`ImageMenuOption`

The menu options for the image.

_Required_: No

_Type_: [ImageMenuOption](aws-properties-quicksight-template-imagemenuoption.md)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
