# AWS::B2BI::Transformer FormatOptions

A structure that contains the X12 transaction set and version.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "X12" : X12Details
}

```

### YAML

```yaml

  X12:
    X12Details

```
