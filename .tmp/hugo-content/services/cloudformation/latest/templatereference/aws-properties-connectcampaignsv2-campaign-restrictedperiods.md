# AWS::ConnectCampaignsV2::Campaign RestrictedPeriods

Contains information about restricted periods.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "RestrictedPeriodList" : [ RestrictedPeriod, ... ]
}

```

### YAML

```yaml

  RestrictedPeriodList:
    - RestrictedPeriod

```

## Properties

`RestrictedPeriodList`

The restricted period list.

_Required_: No

_Type_: Array of [RestrictedPeriod](aws-properties-connectcampaignsv2-campaign-restrictedperiod.md)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
