# AWS::SES::ConfigurationSet ValidationOptions

Specifies the configuration settings for email automatic validation.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "ConditionThreshold" : ConditionThreshold
}

```

### YAML

```yaml

  ConditionThreshold:
    ConditionThreshold

```

## Properties

`ConditionThreshold`

The threshold configuration for the automatic validation settings.

_Required_: Yes

_Type_: [ConditionThreshold](aws-properties-ses-configurationset-conditionthreshold.md)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
