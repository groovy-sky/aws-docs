# AWS::AppIntegrations::Application IframeConfig

The `IframeConfig` property type specifies Property description not available. for an [AWS::AppIntegrations::Application](aws-resource-appintegrations-application.md).

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "Allow" : [ String, ... ],
  "Sandbox" : [ String, ... ]
}

```

### YAML

```yaml

  Allow:
    - String
  Sandbox:
    - String

```
