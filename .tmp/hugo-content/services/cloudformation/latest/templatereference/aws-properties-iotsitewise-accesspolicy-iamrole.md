# AWS::IoTSiteWise::AccessPolicy IamRole

Contains information about an AWS Identity and Access Management role. For more information, see [IAM roles](../../../iam/latest/userguide/id-roles.md) in the
_IAM User Guide_.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "arn" : String
}

```

### YAML

```yaml

  arn: String

```

## Properties

`arn`

The ARN of the IAM role. For more information, see [IAM ARNs](../../../iam/latest/userguide/reference-identifiers.md) in the
_IAM User Guide_.

_Required_: No

_Type_: String

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
