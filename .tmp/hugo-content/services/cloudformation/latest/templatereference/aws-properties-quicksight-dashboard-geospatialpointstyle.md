# AWS::QuickSight::Dashboard GeospatialPointStyle

The point style for a point layer.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "CircleSymbolStyle" : GeospatialCircleSymbolStyle
}

```

### YAML

```yaml

  CircleSymbolStyle:
    GeospatialCircleSymbolStyle

```

## Properties

`CircleSymbolStyle`

The circle symbol style for a point layer.

_Required_: No

_Type_: [GeospatialCircleSymbolStyle](aws-properties-quicksight-dashboard-geospatialcirclesymbolstyle.md)
