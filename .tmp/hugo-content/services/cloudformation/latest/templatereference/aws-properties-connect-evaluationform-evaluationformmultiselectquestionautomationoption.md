# AWS::Connect::EvaluationForm EvaluationFormMultiSelectQuestionAutomationOption

An automation option for a multi-select question.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "RuleCategory" : MultiSelectQuestionRuleCategoryAutomation
}

```

### YAML

```yaml

  RuleCategory:
    MultiSelectQuestionRuleCategoryAutomation

```

## Properties

`RuleCategory`

Rule category configuration for this automation option.

_Required_: Yes

_Type_: [MultiSelectQuestionRuleCategoryAutomation](aws-properties-connect-evaluationform-multiselectquestionrulecategoryautomation.md)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
