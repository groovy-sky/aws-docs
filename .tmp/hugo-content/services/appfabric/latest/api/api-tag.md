# Tag

The key or keys of the key-value pairs for the tag or tags assigned to a
resource.

## Contents

**key**

Tag key.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 128.

Required: Yes

**value**

Tag value.

Type: String

Length Constraints: Minimum length of 0. Maximum length of 256.

Required: Yes
