# FirehoseStream

Contains information about an Amazon Data Firehose delivery stream.

## Contents

**streamName**

The name of the Amazon Data Firehose delivery stream.

Type: String

Length Constraints: Minimum length of 3. Maximum length of 64.

Required: Yes
