# Authorize

The AWS AppFabric for productivity feature is in preview and is subject to change.

Authorizes an AppClient.

###### Topics

- [Request body](#API_Authorize_request)

## Request body

The request accepts the following data in JSON format.

ParameterDescription

**app\_client\_id**

The ID of the AppClient to authorize.

**redirect\_uri**

The URI to redirect end users to after
authorization.

**state**

A unique value to maintain the state between the request
and callback.
