# Using Contributor Insights built-in rules in CloudWatch

You can use Contributor Insights built-in rules
to analyze metrics
from other AWS services.
The following services support built-in rules:

- [Contributor Insights for Amazon DynamoDB](../../../dynamodb/latest/developerguide/contributorinsights.md) in the _Amazon DynamoDB Developer Guide_.

- [Use built-in Contributor Insights rules](../../../vpc/latest/privatelink/privatelink-cloudwatch-metrics.md#privatelink-contributor-insights) in the _AWS PrivateLink Guide_.
