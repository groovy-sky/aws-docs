# Metrics Insights glossary

**label**

In Metrics Insights, a label is a key-value pair that is used to scope a query to
return a particular set of data, or to define criteria by which query results are to
be separated into separate time series. A label key is similar to a column name in
SQL. Labels must be CloudWatch metric dimensions.

**observation**

An observation is a value recorded for a given metric at a given time.
