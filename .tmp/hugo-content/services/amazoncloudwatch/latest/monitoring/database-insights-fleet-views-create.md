# Create a fleet health view for CloudWatch Database Insights

To create a fleet health view, use the following procedure.

###### To create a fleet health view

1. Sign in to the AWS Management Console and open the CloudWatch console at
    [https://console.aws.amazon.com/cloudwatch/](https://console.aws.amazon.com/cloudwatch).

2. Choose **Insights**.

3. Choose **Database Insights**.

4. Choose filters for the fleet of databases you want to monitor.

5. Choose **Save filter as fleet**.

6. In the **Save filter set (fleet)** window, enter a name for your fleet.

7. Choose the **Save** button.

To access the saved fleet, choose the **Saved fleets** dropdown. Then, choose a fleet.
