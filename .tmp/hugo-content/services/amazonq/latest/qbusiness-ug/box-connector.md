# Connecting Box to Amazon Q Business

Box is a cloud storage service that offers file hosting capabilities. You
can connect your Box instance to Amazon Q Business—using either
the AWS Management Console, CLI, or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
