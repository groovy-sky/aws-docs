# Connecting Dropbox to Amazon Q Business

Dropbox is a file hosting service that offers cloud storage, document
organization, and document templating services. You can connect Dropbox instance
to Amazon Q Business—using either the AWS Management Console or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
