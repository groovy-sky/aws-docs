# Connecting SharePoint (Online) to Amazon Q Business

Microsoft SharePoint is a collaborative website building service that lets
you customize web content and create web pages, web sites, document libraries, and lists.
You can connect SharePoint (Online) instance to Amazon Q Business—using either
the AWS Management Console or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
