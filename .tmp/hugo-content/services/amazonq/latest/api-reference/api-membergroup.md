# MemberGroup

The sub groups that belong to a group.

## Contents

**groupName**

The name of the sub group.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 1024.

Pattern: `\P{C}*`

Required: Yes

**type**

The type of the sub group.

Type: String

Valid Values: `INDEX | DATASOURCE`

Required: No
