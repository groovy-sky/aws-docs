# AttachmentInputEvent

A file input event activated by a end user request to upload files into their web
experience chat.

## Contents

**attachment**

This is either a file directly uploaded into a web experience chat or a reference to an existing attachment that is part of a web experience chat.

Type: [AttachmentInput](api-attachmentinput.md) object

Required: No
