# IAM Identity Center users in the AWS Management Console

AWS IAM Identity Center users must change their password from an AWS access portal. For more information, see [Resetting your AWS IAM Identity Center user password](../../../singlesignon/latest/userguide/resetpassword-accessportal.md) in the _AWS IAM Identity Center User Guide_.

An IAM Identity Center user is a user whose AWS account is part of AWS Organizations who signs in through the AWS access portal with a unique URL. These users can
be either created directly in the users in IAM Identity Center or in Active directory or another external identity provider. For more information, see [AWS IAM Identity Center user](../../../signin/latest/userguide/ssolong-user-type.md)
in the _AWS Sign-In User Guide_.
