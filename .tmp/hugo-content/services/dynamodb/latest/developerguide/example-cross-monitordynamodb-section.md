# Monitor performance of Amazon DynamoDB using an AWS SDK

The following code example shows how to configure an application's use of DynamoDB to monitor performance.

Java

**SDK for Java 2.x**

This example shows how to configure a Java application to monitor the performance of DynamoDB. The
application sends metric data to CloudWatch where you can monitor the performance.

For complete source code and instructions on how to set up and run, see the full example on
[GitHub](https://github.com/awsdocs/aws-doc-sdk-examples/tree/main/javav2/usecases/monitor_dynamodb).

###### Services used in this example

- CloudWatch

- DynamoDB
