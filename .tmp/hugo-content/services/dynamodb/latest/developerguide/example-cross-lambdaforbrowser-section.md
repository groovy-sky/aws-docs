# Invoke a Lambda function from a browser

The following code example shows how to invoke an AWS Lambda function from a browser.

JavaScript

**SDK for JavaScript (v2)**

You can create a browser-based application that uses an AWS Lambda function to update an Amazon DynamoDB
table with user selections.

For complete source code and instructions on how to set up and run, see the full example on
[GitHub](https://github.com/awsdocs/aws-doc-sdk-examples/tree/main/javascript/example_code/lambda/lambda-for-browser).

###### Services used in this example

- DynamoDB

- Lambda

**SDK for JavaScript (v3)**

You can create a browser-based application that uses an AWS Lambda function to update an Amazon DynamoDB
table with user selections. This app uses AWS SDK for JavaScript v3.

For complete source code and instructions on how to set up and run, see the full example on
[GitHub](https://github.com/awsdocs/aws-doc-sdk-examples/tree/main/javascriptv3/example_code/cross-services/lambda-for-browser).

###### Services used in this example
