# AWS Identity and Access Management (IAM) and DynamoDB

AWS Identity and Access Management is an AWS service that helps an administrator securely control access to AWS resources. Administrators control
who can be authenticated (signed in) and authorized (have permissions) to use Amazon DynamoDB and DynamoDB Accelerator resources. You can use IAM to manage
access permissions and implement security policies for both Amazon DynamoDB and DynamoDB Accelerator. IAM is an AWS service that you can use with no
additional charge.
