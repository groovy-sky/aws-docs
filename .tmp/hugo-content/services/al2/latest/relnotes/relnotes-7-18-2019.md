# Amazon Linux 2 07/18/2019 release notes

These are release notes for Amazon Linux 2.

## Major updates

An AWS-optimized Linux kernel 4.19 is now available in Amazon Linux 2 Extras channel in addition to the 4.14 kernel that receives long-term
support. You can install and use the 4.19 kernel by running `sudo amazon-linux-extras install kernel-ng` and rebooting your
instance.
