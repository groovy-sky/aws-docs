# Amazon Linux 2 11/19/2018 release notes

These are release notes for Amazon Linux 2.

## Major updates

An ENA driver update that introduces Low Latency Queues (LLQ) for improved average and tail latencies. The update also adds support for
receive checksum offload that improves CPU utilization.
