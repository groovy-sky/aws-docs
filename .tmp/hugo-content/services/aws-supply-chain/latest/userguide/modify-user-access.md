# Modifying existing user access

To modify existing user access, follow these steps:

1. Choose **Settings**, **Users and Permissions**,
    and **Users**.

2. From the **Permission Role** drop-down menu, select the
    appropriate role.

###### Note

Users can have only one permission role. For multiple access privileges, create
a custom role.
