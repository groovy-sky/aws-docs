# S3ExportSpecification

This specification contains a required string of the
destination bucket; optionally, you can include the
destination prefix.

## Contents

**DestinationBucket**

This specifies the destination Amazon S3
bucket for the export job.

Type: String

Required: Yes

**DestinationPrefix**

This specifies the prefix for the destination
Amazon S3 bucket for the export job.

Type: String

Required: No
