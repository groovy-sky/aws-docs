# Stop a backup job

You can stop a backup job in AWS Backup after it has been initiated. When you do this, the
backup is not created, and the backup job record is retained with the status of
**aborted**.

###### To stop a backup job using the AWS Backup console

1. Sign in to the AWS Management Console, and open the AWS Backup console at [https://console.aws.amazon.com/backup](https://console.aws.amazon.com/backup).

2. In the navigation pane on the left, choose **Jobs**.

3. Choose the backup job that you want to stop.

4. In the backup job details pane, choose **Stop**.
