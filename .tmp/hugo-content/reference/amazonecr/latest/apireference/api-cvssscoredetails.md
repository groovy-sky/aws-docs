# CvssScoreDetails

Information about the CVSS score.

## Contents

**adjustments**

An object that contains details about adjustment Amazon Inspector made to the CVSS score.

Type: Array of [CvssScoreAdjustment](api-cvssscoreadjustment.md) objects

Required: No

**score**

The CVSS score.

Type: Double

Required: No

**scoreSource**

The source for the CVSS score.

Type: String

Required: No

**scoringVector**

The vector for the CVSS score.

Type: String

Required: No

**version**

The CVSS version used in scoring.

Type: String

Required: No
