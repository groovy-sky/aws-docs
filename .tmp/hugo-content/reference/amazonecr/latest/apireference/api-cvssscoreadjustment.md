# CvssScoreAdjustment

Details on adjustments Amazon Inspector made to the CVSS score for a finding.

## Contents

**metric**

The metric used to adjust the CVSS score.

Type: String

Required: No

**reason**

The reason the CVSS score has been adjustment.

Type: String

Required: No
