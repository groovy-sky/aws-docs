# CvssScore

The CVSS score for a finding.

## Contents

**baseScore**

The base CVSS score used for the finding.

Type: Double

Required: No

**scoringVector**

The vector string of the CVSS score.

Type: String

Required: No

**source**

The source of the CVSS score.

Type: String

Required: No

**version**

The version of CVSS used for the score.

Type: String

Required: No
