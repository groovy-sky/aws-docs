# UnusedAccessTypeStatistics

Contains information about the total number of findings for a type of unused
access.

## Contents

**total**

The total number of findings for the specified unused access type.

Type: Integer

Required: No

**unusedAccessType**

The type of unused access.

Type: String

Required: No
