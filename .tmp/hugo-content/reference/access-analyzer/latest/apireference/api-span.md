# Span

A span in a policy. The span consists of a start position (inclusive) and end position
(exclusive).

## Contents

**end**

The end position of the span (exclusive).

Type: [Position](api-position.md) object

Required: Yes

**start**

The start position of the span (inclusive).

Type: [Position](api-position.md) object

Required: Yes
