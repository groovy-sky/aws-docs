# JobError

Contains the details about the policy generation error.

## Contents

**code**

The job error code.

Type: String

Valid Values: `AUTHORIZATION_ERROR | RESOURCE_NOT_FOUND_ERROR | SERVICE_QUOTA_EXCEEDED_ERROR | SERVICE_ERROR`

Required: Yes

**message**

Specific information about the error. For example, which service quota was exceeded or
which resource was not found.

Type: String

Required: Yes
