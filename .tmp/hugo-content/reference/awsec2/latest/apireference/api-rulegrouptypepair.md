# RuleGroupTypePair

Describes the type of a stateful rule group.

## Contents

**ruleGroupArn**

The ARN of the rule group.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 1283.

Required: No

**ruleGroupType**

The rule group type. The possible values are `Domain List` and `Suricata`.

Type: String

Required: No
