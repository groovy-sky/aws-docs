# EncryptionSupport

Describes the encryption support status for a transit gateway.

## Contents

**encryptionState**

The current encryption state of the resource.

Type: String

Valid Values: `enabling | enabled | disabling | disabled`

Required: No

**stateMessage**

A message describing the encryption state.

Type: String

Required: No
