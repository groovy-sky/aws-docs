# LoadBalancersConfig

Describes the Classic Load Balancers and target groups to attach to a Spot Fleet
request.

Required: No

**TargetGroupsConfig** (request), **targetGroupsConfig** (response)

The target groups.

Type: [TargetGroupsConfig](api-targetgroupsconfig.md) object

Required: No
