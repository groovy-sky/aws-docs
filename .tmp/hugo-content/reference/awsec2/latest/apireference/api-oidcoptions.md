# OidcOptions

Describes the options for an OpenID Connect-compatible user-identity trust
provider.

## Contents

**authorizationEndpoint**

The OIDC authorization endpoint.

Type: String

Required: No

**clientId**

The client identifier.

Type: String

Required: No

**clientSecret**

The client secret.

Type: String

Required: No

**issuer**

The OIDC issuer.

Type: String

Required: No

**scope**

The OpenID Connect (OIDC) scope specified.

Type: String

Required: No

**tokenEndpoint**

The OIDC token endpoint.

Type: String

Required: No

**userInfoEndpoint**

The OIDC user info endpoint.

Type: String

Required: No
