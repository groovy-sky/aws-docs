# EnaSrdUdpSpecificationRequest

Configures ENA Express for UDP network traffic from your launch template.

## Contents

**EnaSrdUdpEnabled** (request), **EnaSrdUdpEnabled** (response)

Indicates whether UDP traffic uses ENA Express for your instance. To ensure that UDP
traffic can use ENA Express when you launch an instance, you must also set **EnaSrdEnabled** in the **EnaSrdSpecificationRequest** to `true`.

Type: Boolean

Required: No
