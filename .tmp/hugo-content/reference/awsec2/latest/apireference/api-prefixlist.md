# PrefixList

Describes prefixes for AWS services.

## Contents

**CidrSet.N**

The IP address range of the AWS service.

Type: Array of strings

Required: No

**prefixListId**

The ID of the prefix.

Type: String

Required: No

**prefixListName**

The name of the prefix.

Type: String

Required: No
