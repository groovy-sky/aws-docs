# ValidationError

The error code and error message that is returned for a parameter or parameter
combination that is not valid when a new launch template or new version of a launch
template is created.

## Contents

**code**

The error code that indicates why the parameter or parameter combination is not valid.
For more information about error codes, see [Error codes](errors-overview.md).

Type: String

Required: No

**message**

The error message that describes why the parameter or parameter combination is not
valid. For more information about error messages, see [Error codes](errors-overview.md).

Type: String

Required: No
