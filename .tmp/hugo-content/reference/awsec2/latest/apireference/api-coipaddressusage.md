# CoipAddressUsage

Describes address usage for a customer-owned address pool.

## Contents

**allocationId**

The allocation ID of the address.

Type: String

Required: No

**awsAccountId**

The AWS account ID.

Type: String

Required: No

**awsService**

The AWS service.

Type: String

Required: No

**coIp**

The customer-owned IP address.

Type: String

Required: No
