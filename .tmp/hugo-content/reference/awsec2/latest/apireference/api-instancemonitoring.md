# InstanceMonitoring

Describes the monitoring of an instance.

## Contents

**instanceId**

The ID of the instance.

Type: String

Required: No

**monitoring**

The monitoring for the instance.

Type: [Monitoring](api-monitoring.md) object

Required: No
