# UserBucket

Describes the Amazon S3 bucket for the disk image.

## Contents

**S3Bucket**

The name of the Amazon S3 bucket where the disk image is located.

Type: String

Required: No

**S3Key**

The file name of the disk image.

Type: String

Required: No
