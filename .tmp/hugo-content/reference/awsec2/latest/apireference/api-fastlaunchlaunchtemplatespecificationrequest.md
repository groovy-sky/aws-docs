# FastLaunchLaunchTemplateSpecificationRequest

Request to create a launch template for a Windows fast launch enabled AMI.

###### Note

Note - You can specify either the `LaunchTemplateName` or the
`LaunchTemplateId`, but not both.

## Contents

**Version**

Specify the version of the launch template that the AMI should use for Windows fast
launch.

Type: String

Required: Yes

**LaunchTemplateId**

Specify the ID of the launch template that the AMI should use for Windows fast
launch.

Type: String

Required: No

**LaunchTemplateName**

Specify the name of the launch template that the AMI should use for Windows fast
launch.

Type: String

Required: No
