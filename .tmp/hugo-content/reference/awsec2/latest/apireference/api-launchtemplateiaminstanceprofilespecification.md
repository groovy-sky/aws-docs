# LaunchTemplateIamInstanceProfileSpecification

Describes an IAM instance profile.

## Contents

**arn**

The Amazon Resource Name (ARN) of the instance profile.

Type: String

Required: No

**name**

The name of the instance profile.

Type: String

Required: No
