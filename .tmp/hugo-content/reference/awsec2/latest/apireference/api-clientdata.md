# ClientData

Describes the client-specific data.

## Contents

**Comment**

A user-defined comment about the disk upload.

Type: String

Required: No

**UploadEnd**

The time that the disk upload ends.

Type: Timestamp

Required: No

**UploadSize**

The size of the uploaded disk image, in GiB.

Type: Double

Required: No

**UploadStart**

The time that the disk upload starts.

Type: Timestamp

Required: No
