# FilterPortRange

Describes a port range.

## Contents

**fromPort**

The first port in the range.

Type: Integer

Valid Range: Minimum value of 0. Maximum value of 65535.

Required: No

**toPort**

The last port in the range.

Type: Integer

Valid Range: Minimum value of 0. Maximum value of 65535.

Required: No
