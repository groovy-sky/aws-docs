# CreateVerifiedAccessEndpointPortRange

Describes the port range for a Verified Access endpoint.

## Contents

**FromPort**

The start of the port range.

Type: Integer

Valid Range: Minimum value of 1. Maximum value of 65535.

Required: No

**ToPort**

The end of the port range.

Type: Integer

Valid Range: Minimum value of 1. Maximum value of 65535.

Required: No
