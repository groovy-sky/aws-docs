# InstanceExportDetails

Describes an instance to export.

## Contents

**instanceId**

The ID of the resource being exported.

Type: String

Required: No

**targetEnvironment**

The target virtualization environment.

Type: String

Valid Values: `citrix | vmware | microsoft`

Required: No
