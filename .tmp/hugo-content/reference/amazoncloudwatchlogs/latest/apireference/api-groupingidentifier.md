# GroupingIdentifier

A key-value pair that identifies how log groups are grouped in aggregate summaries.

## Contents

**key**

The key that identifies the grouping characteristic. The format of the key uses dot
notation. Examples are, `dataSource.Name`, `dataSource.Type`, and
`dataSource.Format`.

Type: String

Required: No

**value**

The value associated with the grouping characteristic. Examples are
`amazon_vpc`, `flow`, and `OCSF`.

Type: String

Required: No
