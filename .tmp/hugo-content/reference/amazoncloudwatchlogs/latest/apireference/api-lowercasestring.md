# LowerCaseString

This processor converts a string to lowercase.

For more information about this processor including examples, see [lowerCaseString](../../../../services/amazoncloudwatch/latest/logs/cloudwatch-logs-transformation.md#CloudWatch-Logs-Transformation-lowerCaseString) in the _CloudWatch Logs User Guide_.

## Contents

**withKeys**

The array caontaining the keys of the fields to convert to lowercase.

Type: Array of strings

Array Members: Minimum number of 1 item. Maximum number of 10 items.

Length Constraints: Minimum length of 1.

Required: Yes
