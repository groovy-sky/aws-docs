# UpperCaseString

This processor converts a string field to uppercase.

For more information about this processor including examples, see [upperCaseString](../../../../services/amazoncloudwatch/latest/logs/cloudwatch-logs-transformation.md#CloudWatch-Logs-Transformation-upperCaseString) in the _CloudWatch Logs User Guide_.

## Contents

**withKeys**

The array of containing the keys of the field to convert to uppercase.

Type: Array of strings

Array Members: Minimum number of 1 item. Maximum number of 10 items.

Length Constraints: Minimum length of 1.

Required: Yes
