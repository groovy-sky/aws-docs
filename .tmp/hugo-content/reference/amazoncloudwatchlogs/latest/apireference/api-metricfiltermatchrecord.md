# MetricFilterMatchRecord

Represents a matched event.

## Contents

**eventMessage**

The raw event data.

Type: String

Length Constraints: Minimum length of 1.

Required: No

**eventNumber**

The event number.

Type: Long

Required: No

**extractedValues**

The values extracted from the event data by the filter.

Type: String to string map

Required: No
