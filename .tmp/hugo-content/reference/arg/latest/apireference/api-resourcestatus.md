# ResourceStatus

A structure that identifies the current group membership status for a resource. Adding
a resource to a resource group is performed asynchronously as a background task. A
`PENDING` status indicates, for this resource, that the process isn't
completed yet.

## Contents

###### Note

In the following list, the required parameters are described first.

**Name**

The current status.

Type: String

Valid Values: `PENDING`

Required: No
