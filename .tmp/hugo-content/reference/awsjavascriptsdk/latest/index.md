# AWS SDK for JavaScript (v2)

[![NPM version](https://img.shields.io/npm/v/aws-sdk.svg)](https://www.npmjs.com/package/aws-sdk)

## 🚫 End-of-support as of **September 8, 2025**

The AWS SDK for JavaScript v2 has reached [end-of-support](https://aws.amazon.com/blogs/developer/announcing-end-of-support-for-aws-sdk-for-javascript-v2) on **September 8, 2025**.
It will no longer receive updates or releases. Previously published versions are available on npm at
[aws-sdk](https://www.npmjs.com/package/aws-sdk), and source code remains on GitHub at [aws/aws-sdk-js](https://github.com/aws/aws-sdk-js).

We recommend that you migrate to [AWS SDK for JavaScript v3](../../sdk-for-javascript/v3/developer-guide/welcome.md), which has been GA
since December 2020. Here is [why and how you should use it](https://aws.amazon.com/blogs/developer/why-and-how-you-should-use-aws-sdk-for-javascript-v3-on-node-js-18). You can try
migration scripts in [aws-sdk-js-codemod](https://www.npmjs.com/package/aws-sdk-js-codemod) to migrate your application from v2 to v3.

To get help with your migration, please follow our general guidelines to
[open an issue](https://github.com/aws/aws-sdk-js-v3/issues/new?assignees=&labels=needs-triage%2Cv2-v3-inconsistency&projects=&template=v2-to-v3-migration.yml&title=MIGRATION+ISSUE%3A+%5BYour+Title+Here%5D).
To give feedback on and report issues in the v3 repo, please refer to
[Giving feedback and contributing](https://github.com/aws/aws-sdk-js-v3).
