# GeneratedPolicyResult

Contains the text for the generated policy and its details.

## Contents

**properties**

A `GeneratedPolicyProperties` object that contains properties of the
generated policy.

Type: [GeneratedPolicyProperties](api-generatedpolicyproperties.md) object

Required: Yes

**generatedPolicies**

The text to use as the content for the new policy. The policy is created using the
[CreatePolicy](https://docs.aws.amazon.com/IAM/latest/APIReference/API_CreatePolicy.html) action.

Type: Array of [GeneratedPolicy](api-generatedpolicy.md) objects

Required: No
