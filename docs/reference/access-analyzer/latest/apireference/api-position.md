# Position

A position in a policy.

## Contents

**column**

The column of the position, starting from 0.

Type: Integer

Required: Yes

**line**

The line of the position, starting from 1.

Type: Integer

Required: Yes

**offset**

The offset within the policy that corresponds to the position, starting from 0.

Type: Integer

Required: Yes
