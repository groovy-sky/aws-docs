# PolicyGenerationDetails

Contains the ARN details about the IAM entity for which the policy is
generated.

## Contents

**principalArn**

The ARN of the IAM entity (user or role) for which you are generating a policy.

Type: String

Pattern: `arn:[^:]*:iam::[^:]*:(role|user)/.{1,576}`

Required: Yes
