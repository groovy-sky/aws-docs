# DataSourceFilter

Filter criteria for data sources, used to specify which data sources to include in
operations based on name and type.

## Contents

**name**

The name pattern to filter data sources by.

Type: String

Required: Yes

**type**

The type pattern to filter data sources by.

Type: String

Required: No
