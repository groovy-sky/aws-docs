# SplitStringEntry

This object defines one log field that will be split with the [splitString](../../../../services/amazoncloudwatch/latest/logs/cloudwatch-logs-transformation.md#CloudWatch-Logs-Transformation-splitString) processor.

## Contents

**delimiter**

The separator characters to split the string entry on.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 128.

Required: Yes

**source**

The key of the field to split.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 128.

Required: Yes
