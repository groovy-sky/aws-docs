# DeleteKeys

This processor deletes entries from a log event. These entries are key-value pairs.

For more information about this processor including examples, see [deleteKeys](../../../../services/amazoncloudwatch/latest/logs/cloudwatch-logs-transformation.md#CloudWatch-Logs-Transformation-deleteKeys) in the _CloudWatch Logs User Guide_.

## Contents

**withKeys**

The list of keys to delete.

Type: Array of strings

Array Members: Minimum number of 1 item. Maximum number of 5 items.

Length Constraints: Minimum length of 1.

Required: Yes
