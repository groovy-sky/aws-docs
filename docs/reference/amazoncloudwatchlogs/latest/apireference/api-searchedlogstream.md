# SearchedLogStream

Represents the search status of a log stream.

## Contents

**logStreamName**

The name of the log stream.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 512.

Pattern: `[^:*]*`

Required: No

**searchedCompletely**

Indicates whether all the events in this log stream were searched.

Type: Boolean

Required: No
