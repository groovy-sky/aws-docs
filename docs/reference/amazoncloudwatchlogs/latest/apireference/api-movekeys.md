# MoveKeys

This processor moves a key from one field to another. The original key is deleted.

For more information about this processor including examples, see [moveKeys](../../../../services/amazoncloudwatch/latest/logs/cloudwatch-logs-transformation.md#CloudWatch-Logs-Transformation-moveKeys) in the _CloudWatch Logs User Guide_.

## Contents

**entries**

An array of objects, where each object contains the information about one key to move.

Type: Array of [MoveKeyEntry](api-movekeyentry.md) objects

Array Members: Minimum number of 1 item. Maximum number of 5 items.

Required: Yes
