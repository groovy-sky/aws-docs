# ExportTaskStatus

Represents the status of an export task.

## Contents

**code**

The status code of the export task.

Type: String

Valid Values: `CANCELLED | COMPLETED | FAILED | PENDING | PENDING_CANCEL | RUNNING`

Required: No

**message**

The status message related to the status code.

Type: String

Required: No
