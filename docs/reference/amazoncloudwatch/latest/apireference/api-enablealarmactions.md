# EnableAlarmActions

Enables the actions for the specified alarms.

## Request Parameters

**AlarmNames**

The names of the alarms.

Type: Array of strings

Array Members: Maximum number of 100 items.

Length Constraints: Minimum length of 1. Maximum length of 255.

Required: Yes
