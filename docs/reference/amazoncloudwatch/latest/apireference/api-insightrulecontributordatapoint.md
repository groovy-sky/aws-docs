# InsightRuleContributorDatapoint

One data point related to one contributor.

For more information, see [GetInsightRuleReport](api-getinsightrulereport.md) and [InsightRuleContributor](api-insightrulecontributor.md).

## Contents

**ApproximateValue**

The approximate value that this contributor added during this timestamp.

Type: Double

Required: Yes

**Timestamp**

The timestamp of the data point.

Type: Timestamp

Required: Yes
