# S3InputFormatConfig

When you use Amazon S3 as the source, the configuration format that you provide
the flow input data.

## Contents

**s3InputFileType**

The file type that Amazon AppFlow gets from your Amazon S3 bucket.

Type: String

Valid Values: `CSV | JSON`

Required: No
