# ServiceNowConnectorProfileProperties

The connector-specific profile properties required when using ServiceNow.

## Contents

**instanceUrl**

The location of the ServiceNow resource.

Type: String

Length Constraints: Maximum length of 256.

Pattern: `\S+`

Required: Yes
