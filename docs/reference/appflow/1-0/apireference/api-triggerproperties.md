# TriggerProperties

Specifies the configuration details that control the trigger for a flow. Currently, these
settings only apply to the `Scheduled` trigger type.

## Contents

**Scheduled**

Specifies the configuration details of a schedule-triggered flow as defined by the user.

Type: [ScheduledTriggerProperties](api-scheduledtriggerproperties.md) object

Required: No
