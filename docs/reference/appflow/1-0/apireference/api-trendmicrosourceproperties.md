# TrendmicroSourceProperties

The properties that are applied when using Trend Micro as a flow source.

## Contents

**object**

The object specified in the Trend Micro flow source.

Type: String

Length Constraints: Maximum length of 512.

Pattern: `\S+`

Required: Yes
