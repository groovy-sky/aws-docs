# ConnectorProfileConfig

Defines the connector-specific configuration and credentials for the connector profile.

## Contents

**connectorProfileProperties**

The connector-specific properties of the profile configuration.

Type: [ConnectorProfileProperties](https://docs.aws.amazon.com/appflow/1.0/APIReference/API_ConnectorProfileProperties.html) object

Required: Yes

**connectorProfileCredentials**

The connector-specific credentials required by each connector.

Type: [ConnectorProfileCredentials](https://docs.aws.amazon.com/appflow/1.0/APIReference/API_ConnectorProfileCredentials.html) object

Required: No
