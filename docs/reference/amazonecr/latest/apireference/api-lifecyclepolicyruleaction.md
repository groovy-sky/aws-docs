# LifecyclePolicyRuleAction

The type of action to be taken.

## Contents

**targetStorageClass**

The target storage class for the action. This is only present when the `type` is `TRANSITION.`

Type: String

Valid Values: `ARCHIVE`

Required: No

**type**

The type of action to be taken.

Type: String

Valid Values: `EXPIRE | TRANSITION`

Required: No
