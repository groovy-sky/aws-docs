# ClusterSummary

A summary of the properties of a cluster.

## Contents

**arn**

The ARN of the cluster.

Type: String

Pattern: `arn:aws(-[^:]+)?:dsql:[a-z0-9-]{1,20}:[0-9]{12}:cluster/[a-z0-9]{26}`

Required: Yes

**identifier**

The ID of the cluster.

Type: String

Pattern: `[a-z0-9]{26}`

Required: Yes
