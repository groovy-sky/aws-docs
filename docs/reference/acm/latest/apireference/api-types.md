# Data Types

The AWS Certificate Manager API contains several data types that various actions use. This section describes each data type in detail.

###### Note

The order of each element in a data type structure is not guaranteed. Applications should not assume a particular order.

The following data types are supported:

- [CertificateDetail](api-certificatedetail.md)

- [CertificateOptions](api-certificateoptions.md)

- [CertificateSummary](api-certificatesummary.md)

- [DomainValidation](api-domainvalidation.md)

- [DomainValidationOption](api-domainvalidationoption.md)

- [ExpiryEventsConfiguration](api-expiryeventsconfiguration.md)

- [ExtendedKeyUsage](api-extendedkeyusage.md)

- [Filters](api-filters.md)

- [HttpRedirect](api-httpredirect.md)

- [KeyUsage](api-keyusage.md)

- [RenewalSummary](api-renewalsummary.md)

- [ResourceRecord](api-resourcerecord.md)

- [Tag](api-tag.md)

- [ThrottlingReason](api-throttlingreason.md)
