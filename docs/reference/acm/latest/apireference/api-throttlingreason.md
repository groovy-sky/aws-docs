# ThrottlingReason

A description of why a request was throttled.

## Contents

###### Note

In the following list, the required parameters are described first.

**reason**

A description of why a request was throttled.

Type: String

Required: No

**resource**

The resource that causes the request to be throttled.

Type: String

Required: No
