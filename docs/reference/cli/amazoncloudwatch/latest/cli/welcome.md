# Welcome

As of November 7, 2017, we are no longer supporting the CloudWatch command line interface with new functionality. It is not available for download. The
CloudWatch CLI reference documentation is still available.

We encourage customers
to use the AWS Command Line Interface. The AWS CLI includes all existing and new CloudWatch commands, and is the only command line interface being updated.

For information about installing the AWS CLI, see [Installing the AWS Command Line Interface](../../../../../services/cli/latest/userguide/installing.md). For information
about CloudWatch commands in the AWS CLI, see [cloudwatch](../../../../../services/cli/latest/reference/cloudwatch/index.md).
