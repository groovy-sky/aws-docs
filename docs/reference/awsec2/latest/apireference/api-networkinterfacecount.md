# NetworkInterfaceCount

The minimum and maximum number of network interfaces.

## Contents

**Max** (request), **max** (response)

The maximum number of network interfaces. If this parameter is not specified, there is no
maximum limit.

Type: Integer

Required: No

**Min** (request), **min** (response)

The minimum number of network interfaces. If this parameter is not specified, there is no
minimum limit.

Type: Integer

Required: No
