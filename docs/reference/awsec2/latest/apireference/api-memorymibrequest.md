# MemoryMiBRequest

The minimum and maximum amount of memory, in MiB.

## Contents

**Min**

The minimum amount of memory, in MiB. To specify no minimum limit, specify
`0`.

Type: Integer

Required: Yes

**Max**

The maximum amount of memory, in MiB. To specify no maximum limit, omit this
parameter.

Type: Integer

Required: No
