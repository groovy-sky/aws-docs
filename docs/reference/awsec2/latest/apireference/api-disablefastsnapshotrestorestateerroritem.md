# DisableFastSnapshotRestoreStateErrorItem

Contains information about an error that occurred when disabling fast snapshot restores.

## Contents

**availabilityZone**

The Availability Zone.

Type: String

Required: No

**availabilityZoneId**

The ID of the Availability Zone.

Type: String

Required: No

**error**

The error.

Type: [DisableFastSnapshotRestoreStateError](api-disablefastsnapshotrestorestateerror.md) object

Required: No
