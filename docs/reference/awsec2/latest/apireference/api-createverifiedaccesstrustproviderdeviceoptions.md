# CreateVerifiedAccessTrustProviderDeviceOptions

Describes the options when creating an AWS Verified Access trust provider using the
`device` type.

## Contents

**PublicSigningKeyUrl**

The URL AWS Verified Access will use to verify the authenticity of the device tokens.

Type: String

Required: No

**TenantId**

The ID of the tenant application with the device-identity provider.

Type: String

Required: No
