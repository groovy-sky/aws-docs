# VpcEncryptionNonCompliantResource

Describes a resource that is not compliant with VPC encryption requirements.

For more information, see [Enforce VPC encryption in transit](../../../../services/vpc/latest/userguide/vpc-encryption-controls.md) in the _Amazon VPC User Guide_.

## Contents

**description**

A description of the non-compliant resource.

Type: String

Required: No

**id**

The ID of the non-compliant resource.

Type: String

Required: No

**isExcludable**

Indicates whether the resource can be excluded from encryption enforcement.

Type: Boolean

Required: No

**type**

The type of the non-compliant resource.

Type: String

Required: No
