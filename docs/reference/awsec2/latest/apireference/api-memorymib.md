# MemoryMiB

The minimum and maximum amount of memory, in MiB.

## Contents

**Max** (request), **max** (response)

The maximum amount of memory, in MiB. If this parameter is not specified, there is no
maximum limit.

Type: Integer

Required: No

**Min** (request), **min** (response)

The minimum amount of memory, in MiB. If this parameter is not specified, there is no minimum
limit.

Type: Integer

Required: No
