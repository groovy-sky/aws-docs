# VpcCidrBlockState

Describes the state of a CIDR block.

## Contents

**state**

The state of the CIDR block.

Type: String

Valid Values: `associating | associated | disassociating | disassociated | failing | failed`

Required: No

**statusMessage**

A message about the status of the CIDR block, if applicable.

Type: String

Required: No
