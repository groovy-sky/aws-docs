# NetworkInterfacePermissionState

Describes the state of a network interface permission.

## Contents

**state**

The state of the permission.

Type: String

Valid Values: `pending | granted | revoking | revoked`

Required: No

**statusMessage**

A status message, if applicable.

Type: String

Required: No
