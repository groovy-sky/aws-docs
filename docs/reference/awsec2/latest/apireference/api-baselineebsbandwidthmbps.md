# BaselineEbsBandwidthMbps

The minimum and maximum baseline bandwidth to Amazon EBS, in Mbps. For more information, see
[Amazon\
EBS–optimized instances](../../../../services/ec2/latest/userguide/ebs-optimized.md) in the _Amazon EC2 User Guide_.

## Contents

**Max** (request), **max** (response)

The maximum baseline bandwidth, in Mbps. If this parameter is not specified, there is no
maximum limit.

Type: Integer

Required: No

**Min** (request), **min** (response)

The minimum baseline bandwidth, in Mbps. If this parameter is not specified, there is no
minimum limit.

Type: Integer

Required: No
