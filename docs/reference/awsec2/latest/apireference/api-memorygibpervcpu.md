# MemoryGiBPerVCpu

The minimum and maximum amount of memory per vCPU, in GiB.

## Contents

**Max** (request), **max** (response)

The maximum amount of memory per vCPU, in GiB. If this parameter is not specified, there is
no maximum limit.

Type: Double

Required: No

**Min** (request), **min** (response)

The minimum amount of memory per vCPU, in GiB. If this parameter is not specified, there is
no minimum limit.

Type: Double

Required: No
