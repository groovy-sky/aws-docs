# RegionalSummary

A summary report for the attribute for a Region.

## Contents

**numberOfMatchedAccounts**

The number of accounts in the Region with the same configuration value for the
attribute that is most frequently observed.

Type: Integer

Required: No

**numberOfUnmatchedAccounts**

The number of accounts in the Region with a configuration value different from the
most frequently observed value for the attribute.

Type: Integer

Required: No

**regionName**

The AWS Region.

Type: String

Required: No
