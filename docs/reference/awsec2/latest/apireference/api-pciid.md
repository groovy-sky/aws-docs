# PciId

Describes the data that identifies an Amazon FPGA image (AFI) on the PCI bus.

## Contents

**DeviceId**

The ID of the device.

Type: String

Required: No

**SubsystemId**

The ID of the subsystem.

Type: String

Required: No

**SubsystemVendorId**

The ID of the vendor for the subsystem.

Type: String

Required: No

**VendorId**

The ID of the vendor.

Type: String

Required: No
