# CapacityReservationGroup

Describes a resource group to which a Capacity Reservation has been added.

## Contents

**groupArn**

The ARN of the resource group.

Type: String

Required: No

**ownerId**

The ID of the AWS account that owns the resource group.

Type: String

Required: No
