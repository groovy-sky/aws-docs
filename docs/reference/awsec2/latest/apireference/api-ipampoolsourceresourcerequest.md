# IpamPoolSourceResourceRequest

The resource used to provision CIDRs to a resource planning pool.

**ResourceId**

The source resource ID.

Type: String

Required: No

**ResourceOwner**

The source resource owner.

Type: String

Required: No

**ResourceRegion**

The source resource Region.

Type: String

Required: No

**ResourceType**

The source resource type.

Type: String

Valid Values: `vpc`

Required: No
