# InstanceStateChange

Describes an instance state change.

## Contents

**currentState**

The current state of the instance.

Type: [InstanceState](api-instancestate.md) object

Required: No

**instanceId**

The ID of the instance.

Type: String

Required: No

**previousState**

The previous state of the instance.

Type: [InstanceState](api-instancestate.md) object

Required: No
