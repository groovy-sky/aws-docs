# NeuronDeviceCoreInfo

Describes the cores available to the neuron accelerator.

## Contents

**count**

The number of cores available to the neuron accelerator.

Type: Integer

Required: No

**version**

The version of the neuron accelerator.

Type: Integer

Required: No
