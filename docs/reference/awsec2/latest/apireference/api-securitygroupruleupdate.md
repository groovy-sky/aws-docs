# SecurityGroupRuleUpdate

Describes an update to a security group rule.

## Contents

**SecurityGroupRuleId**

The ID of the security group rule.

Type: String

Required: Yes

**SecurityGroupRule**

Information about the security group rule.

Type: [SecurityGroupRuleRequest](api-securitygrouprulerequest.md) object

Required: No
