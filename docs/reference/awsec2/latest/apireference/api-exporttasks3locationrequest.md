# ExportTaskS3LocationRequest

Describes the destination for an export image task.

## Contents

**S3Bucket**

The destination Amazon S3 bucket.

Type: String

Required: Yes

**S3Prefix**

The prefix (logical hierarchy) in the bucket.

Type: String

Required: No
