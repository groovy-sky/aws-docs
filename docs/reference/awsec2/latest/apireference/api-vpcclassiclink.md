# VpcClassicLink

###### Note

Deprecated.

Describes whether a VPC is enabled for ClassicLink.

## Contents

**classicLinkEnabled**

Indicates whether the VPC is enabled for ClassicLink.

Type: Boolean

Required: No

**TagSet.N**

Any tags assigned to the VPC.

Type: Array of [Tag](api-tag.md) objects

Required: No

**vpcId**

The ID of the VPC.

Type: String

Required: No
