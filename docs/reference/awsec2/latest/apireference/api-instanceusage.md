# InstanceUsage

Information about the Capacity Reservation usage.

## Contents

**accountId**

The ID of the AWS account that is making use of the Capacity
Reservation.

Type: String

Required: No

**usedInstanceCount**

The number of instances the AWS account currently has in the Capacity
Reservation.

Type: Integer

Required: No
