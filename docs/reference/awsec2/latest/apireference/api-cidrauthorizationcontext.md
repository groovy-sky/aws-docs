# CidrAuthorizationContext

Provides authorization for Amazon to bring a specific IP address range to a specific
AWS account using bring your own IP addresses (BYOIP). For more information, see [Configuring your BYOIP address range](../../../../services/ec2/latest/userguide/ec2-byoip.md#prepare-for-byoip) in the _Amazon EC2 User Guide_.

## Contents

**Message**

The plain-text authorization message for the prefix and account.

Type: String

Required: Yes

**Signature**

The signed authorization message for the prefix and account.

Type: String

Required: Yes
