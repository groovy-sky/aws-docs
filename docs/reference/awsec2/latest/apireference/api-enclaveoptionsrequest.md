# EnclaveOptionsRequest

Indicates whether the instance is enabled for AWS Nitro Enclaves. For
more information, see [What is AWS Nitro\
Enclaves?](https://docs.aws.amazon.com/enclaves/latest/user/nitro-enclave.html) in the _AWS Nitro Enclaves User_
_Guide_.

## Contents

**Enabled**

To enable the instance for AWS Nitro Enclaves, set this parameter to
`true`.

Type: Boolean

Required: No
