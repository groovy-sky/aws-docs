# UserBucketDetails

Describes the Amazon S3 bucket for the disk image.

## Contents

**s3Bucket**

The Amazon S3 bucket from which the disk image was created.

Type: String

Required: No

**s3Key**

The file name of the disk image.

Type: String

Required: No
