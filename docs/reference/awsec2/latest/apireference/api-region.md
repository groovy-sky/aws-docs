# Region

Describes a Region.

## Contents

**GeographySet.N**

The geography information for the Region. The geography is returned as a list.

Type: Array of [RegionGeography](https://docs.aws.amazon.com/AWSEC2/latest/APIReference/API_RegionGeography.html) objects

Required: No

**optInStatus**

The Region opt-in status. The possible values are `opt-in-not-required`, `opted-in`, and
`not-opted-in`.

Type: String

Required: No

**regionEndpoint**

The Region service endpoint.

Type: String

Required: No

**regionName**

The name of the Region.

Type: String

Required: No
