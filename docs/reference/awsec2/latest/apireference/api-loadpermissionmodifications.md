# LoadPermissionModifications

Describes modifications to the load permissions of an Amazon FPGA image (AFI).

## Contents

**Add.N**

The load permissions to add.

Type: Array of [LoadPermissionRequest](https://docs.aws.amazon.com/AWSEC2/latest/APIReference/API_LoadPermissionRequest.html) objects

Required: No

**Remove.N**

The load permissions to remove.

Type: Array of [LoadPermissionRequest](https://docs.aws.amazon.com/AWSEC2/latest/APIReference/API_LoadPermissionRequest.html) objects

Required: No
