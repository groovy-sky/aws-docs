# CreateVolumePermission

Describes the user or group to be added or removed from the list of create volume
permissions for a volume.

## Contents

**Group** (request), **group** (response)

The group to be added or removed. The possible value is `all`.

Type: String

Valid Values: `all`

Required: No

**UserId** (request), **userId** (response)

The ID of the AWS account to be added or removed.

Type: String

Required: No
