# NetworkInsightsAccessScopeContent

Describes the Network Access Scope content.

## Contents

**ExcludePathSet.N**

The paths to exclude.

Type: Array of [AccessScopePath](https://docs.aws.amazon.com/AWSEC2/latest/APIReference/API_AccessScopePath.html) objects

Required: No

**MatchPathSet.N**

The paths to match.

Type: Array of [AccessScopePath](https://docs.aws.amazon.com/AWSEC2/latest/APIReference/API_AccessScopePath.html) objects

Required: No

**networkInsightsAccessScopeId**

The ID of the Network Access Scope.

Type: String

Required: No
