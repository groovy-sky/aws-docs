# Monitoring

Describes the monitoring of an instance.

## Contents

**state**

Indicates whether detailed monitoring is enabled. Otherwise, basic monitoring is
enabled.

Type: String

Valid Values: `disabled | disabling | enabled | pending`

Required: No
