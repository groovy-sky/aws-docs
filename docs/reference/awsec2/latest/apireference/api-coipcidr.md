# CoipCidr

Information about a customer-owned IP address range.

## Contents

**cidr**

An address range in a customer-owned IP address space.

Type: String

Required: No

**coipPoolId**

The ID of the address pool.

Type: String

Required: No

**localGatewayRouteTableId**

The ID of the local gateway route table.

Type: String

Required: No
