# ValidationWarning

The error codes and error messages that are returned for the parameters or parameter
combinations that are not valid when a new launch template or new version of a launch
template is created.

## Contents

**ErrorSet.N**

The error codes and error messages.

Type: Array of [ValidationError](https://docs.aws.amazon.com/AWSEC2/latest/APIReference/API_ValidationError.html) objects

Required: No
