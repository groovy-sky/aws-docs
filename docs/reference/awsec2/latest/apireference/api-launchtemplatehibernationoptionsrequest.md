# LaunchTemplateHibernationOptionsRequest

Indicates whether the instance is configured for hibernation. This parameter is valid
only if the instance meets the [hibernation\
prerequisites](../../../../services/ec2/latest/userguide/hibernating-prerequisites.md).

## Contents

**Configured**

If you set this parameter to `true`, the instance is enabled for
hibernation.

Default: `false`

Type: Boolean

Required: No
