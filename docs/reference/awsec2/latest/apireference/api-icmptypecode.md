# IcmpTypeCode

Describes the ICMP type and code.

## Contents

**Code** (request), **code** (response)

The ICMP code. A value of -1 means all codes for the specified ICMP type.

Type: Integer

Required: No

**Type** (request), **type** (response)

The ICMP type. A value of -1 means all types.

Type: Integer

Required: No
