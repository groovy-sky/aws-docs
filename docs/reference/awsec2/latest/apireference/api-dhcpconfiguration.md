# DhcpConfiguration

Describes a DHCP configuration option.

## Contents

**key**

The name of a DHCP option.

Type: String

Required: No

**ValueSet.N**

The values for the DHCP option.

Type: Array of [AttributeValue](api-attributevalue.md) objects

Required: No
