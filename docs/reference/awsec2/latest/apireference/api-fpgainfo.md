# FpgaInfo

Describes the FPGAs for the instance type.

## Contents

**Fpgas.N**

Describes the FPGAs for the instance type.

Type: Array of [FpgaDeviceInfo](api-fpgadeviceinfo.md) objects

Required: No

**totalFpgaMemoryInMiB**

The total memory of all FPGA accelerators for the instance type.

Type: Integer

Required: No
