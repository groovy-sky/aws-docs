# AsnAuthorizationContext

Provides authorization for Amazon to bring an Autonomous System Number (ASN) to a specific AWS account using bring your own ASN (BYOASN).
For details on the format of the message and signature, see [Tutorial: Bring your ASN to IPAM](https://docs.aws.amazon.com/vpc/latest/ipam/tutorials-byoasn.html) in the _Amazon VPC IPAM guide_.

## Contents

**Message**

The authorization context's message.

Type: String

Required: Yes

**Signature**

The authorization context's signature.

Type: String

Required: Yes
