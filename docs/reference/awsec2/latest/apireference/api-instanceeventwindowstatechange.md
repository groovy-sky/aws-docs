# InstanceEventWindowStateChange

The state of the event window.

## Contents

**instanceEventWindowId**

The ID of the event window.

Type: String

Required: No

**state**

The current state of the event window.

Type: String

Valid Values: `creating | deleting | active | deleted`

Required: No
