# AnalysisComponent

Describes a path component.

## Contents

**arn**

The Amazon Resource Name (ARN) of the component.

Type: String

Required: No

**id**

The ID of the component.

Type: String

Required: No

**name**

The name of the analysis component.

Type: String

Required: No
