# FastLaunchSnapshotConfigurationRequest

Configuration settings for creating and managing pre-provisioned snapshots for a Windows
fast launch enabled AMI.

## Contents

**TargetResourceCount**

The number of pre-provisioned snapshots to keep on hand for a Windows fast launch enabled
AMI.

Type: Integer

Required: No
