# DiskInfo

Describes a disk.

## Contents

**count**

The number of disks with this configuration.

Type: Integer

Required: No

**sizeInGB**

The size of the disk in GB.

Type: Long

Required: No

**type**

The type of disk.

Type: String

Valid Values: `hdd | ssd`

Required: No
