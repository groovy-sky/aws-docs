# EnclaveOptions

Indicates whether the instance is enabled for AWS Nitro
Enclaves.

## Contents

**enabled**

If this parameter is set to `true`, the instance is enabled for AWS Nitro Enclaves; otherwise, it is not enabled for AWS Nitro
Enclaves.

Type: Boolean

Required: No
