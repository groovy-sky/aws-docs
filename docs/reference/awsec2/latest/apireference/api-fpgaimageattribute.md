# FpgaImageAttribute

Describes an Amazon FPGA image (AFI) attribute.

## Contents

**description**

The description of the AFI.

Type: String

Required: No

**fpgaImageId**

The ID of the AFI.

Type: String

Required: No

**LoadPermissions.N**

The load permissions.

Type: Array of [LoadPermission](https://docs.aws.amazon.com/AWSEC2/latest/APIReference/API_LoadPermission.html) objects

Required: No

**name**

The name of the AFI.

Type: String

Required: No

**ProductCodes.N**

The product codes.

Type: Array of [ProductCode](api-productcode.md) objects

Required: No
