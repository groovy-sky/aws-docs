# SupportedRegionDetail

Describes a supported Region.

## Contents

**region**

The Region code.

Type: String

Required: No

**serviceState**

The service state. The possible values are `Pending`, `Available`,
`Deleting`, `Deleted`, `Failed`, and `Closed`.

Type: String

Required: No
