# ScheduledInstancesPrivateIpAddressConfig

Describes a private IPv4 address for a Scheduled Instance.

## Contents

**Primary**

Indicates whether this is a primary IPv4 address. Otherwise, this is a secondary IPv4 address.

Type: Boolean

Required: No

**PrivateIpAddress**

The IPv4 address.

Type: String

Required: No
