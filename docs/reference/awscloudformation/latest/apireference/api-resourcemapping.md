# ResourceMapping

Specifies the current source of the resource and the destination of where it will be moved
to.

## Contents

**Destination**

The destination stack `StackName` and `LogicalResourceId` for the
resource being refactored.

Type: [ResourceLocation](api-resourcelocation.md) object

Required: Yes

**Source**

The source stack `StackName` and `LogicalResourceId` for the resource
being refactored.

Type: [ResourceLocation](api-resourcelocation.md) object

Required: Yes
