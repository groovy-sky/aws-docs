# ServiceEvent

The details for an event that's associated with a service.

## Contents

**createdAt**

The Unix timestamp for the time when the event was triggered.

Type: Timestamp

Required: No

**id**

The ID string for the event.

Type: String

Required: No

**message**

The event message.

Type: String

Required: No
