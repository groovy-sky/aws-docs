# TimeRangeFilter

Filters update actions from the service updates that are in available status during
the time range.

## Contents

###### Note

In the following list, the required parameters are described first.

**EndTime**

The end time of the time range filter

Type: Timestamp

Required: No

**StartTime**

The start time of the time range filter

Type: Timestamp

Required: No
