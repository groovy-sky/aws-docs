# ParameterNameValue

Describes a name-value pair that is used to update the value of a parameter.

## Contents

###### Note

In the following list, the required parameters are described first.

**ParameterName**

The name of the parameter.

Type: String

Required: No

**ParameterValue**

The value of the parameter.

Type: String

Required: No
