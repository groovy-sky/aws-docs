# CapacityAssignment

A mapping between one or more workgroups and a capacity reservation.

## Contents

**WorkGroupNames**

The list of workgroup names for the capacity assignment.

Type: Array of strings

Pattern: `[a-zA-Z0-9._-]{1,128}`

Required: No
