# ResultSet

The metadata and rows that make up a query result set. The metadata describes the
column structure and data types. To return a `ResultSet` object, use [GetQueryResults](api-getqueryresults.md).

## Contents

**ResultSetMetadata**

The metadata that describes the column structure and data types of a table of query
results.

Type: [ResultSetMetadata](api-resultsetmetadata.md) object

Required: No

**Rows**

The rows in the table.

Type: Array of [Row](api-row.md) objects

Required: No
