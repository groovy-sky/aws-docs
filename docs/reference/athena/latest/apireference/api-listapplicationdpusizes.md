# ListApplicationDPUSizes

Returns the supported DPU sizes for the supported application runtimes (for example,
`Athena notebook version 1`).

## Request Syntax

```nohighlight

{
   "MaxResults": number,
   "NextToken": "string"
}
```

## Request Parameters

For information about the parameters that are common to all actions, see [Common Parameters](https://docs.aws.amazon.com/athena/latest/APIReference/CommonParameters.html).

The request accepts the following data in JSON format.

**[MaxResults](#API_ListApplicationDPUSizes_RequestSyntax)**

Specifies the maximum number of results to return.

Type: Integer

Valid Range: Minimum value of 1. Maximum value of 100.

Required: No

**[NextToken](#API_ListApplicationDPUSizes_RequestSyntax)**

A token generated by the Athena service that specifies where to continue
pagination if a previous request was truncated.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 1024.

Required: No

## Response Syntax

```nohighlight

{
   "ApplicationDPUSizes": [
      {
         "ApplicationRuntimeId": "string",
         "SupportedDPUSizes": [ number ]
      }
   ],
   "NextToken": "string"
}
```

## Response Elements

If the action is successful, the service sends back an HTTP 200 response.

The following data is returned in JSON format by the service.

**[ApplicationDPUSizes](#API_ListApplicationDPUSizes_ResponseSyntax)**

A list of the supported DPU sizes that the application runtime supports.

Type: Array of [ApplicationDPUSizes](https://docs.aws.amazon.com/athena/latest/APIReference/API_ApplicationDPUSizes.html) objects

**[NextToken](#API_ListApplicationDPUSizes_ResponseSyntax)**

A token generated by the Athena service that specifies where to continue
pagination if a previous request was truncated. To obtain the next set of pages, pass in
the `NextToken` from the response object of the previous page call.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 1024.
