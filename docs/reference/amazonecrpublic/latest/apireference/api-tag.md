# Tag

The metadata that you apply to a resource to help you categorize and organize them. Each
tag consists of a key and an optional value. You define both. Tag keys can have a maximum character length of 128 characters, and tag values can have a maximum length of 256 characters.

## Contents

**Key**

One part of a key-value pair that make up a tag. A `key` is a general label
that acts like a category for more specific tag values.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 128.

Required: No

**Value**

The optional part of a key-value pair that make up a tag. A `value` acts as a
descriptor within a tag category (key).

Type: String

Length Constraints: Minimum length of 0. Maximum length of 256.

Required: No
