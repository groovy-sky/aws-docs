# AttributeValue

The values of a given attribute, such as `Throughput Optimized HDD` or `Provisioned
          IOPS` for the `Amazon EC2` `volumeType` attribute.

## Contents

**Value**

The specific value of an `attributeName`.

Type: String

Required: No
