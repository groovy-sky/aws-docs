# Tag

Metadata that help you categorize IT assets.

###### Important

Do not store sensitive information (like personal data) in tags.

## Contents

**key**

The type of tag on which to filter.

Type: String

Required: Yes

**value**

A value for a tag key on which to filter.

Type: String

Required: Yes
