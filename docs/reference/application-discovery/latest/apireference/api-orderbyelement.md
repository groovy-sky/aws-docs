# OrderByElement

A field and direction for ordered output.

## Contents

**fieldName**

The field on which to order.

Type: String

Length Constraints: Maximum length of 1000.

Pattern: `[\s\S]*\S[\s\S]*`

Required: Yes

**sortOrder**

Ordering direction.

Type: String

Valid Values: `ASC | DESC`

Required: No
