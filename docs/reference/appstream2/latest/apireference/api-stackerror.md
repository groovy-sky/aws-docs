# StackError

Describes a stack error.

## Contents

**ErrorCode**

The error code.

Type: String

Valid Values: `STORAGE_CONNECTOR_ERROR | INTERNAL_SERVICE_ERROR`

Required: No

**ErrorMessage**

The error message.

Type: String

Length Constraints: Minimum length of 1.

Required: No
