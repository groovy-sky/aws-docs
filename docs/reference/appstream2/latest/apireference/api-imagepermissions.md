# ImagePermissions

Describes the permissions for an image.

## Contents

**allowFleet**

Indicates whether the image can be used for a fleet.

Type: Boolean

Required: No

**allowImageBuilder**

Indicates whether the image can be used for an image builder.

Type: Boolean

Required: No
