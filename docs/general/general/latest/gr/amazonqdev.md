# Amazon Q Developer endpoints and quotas

Service quotas, also referred to as limits, are the maximum number of service resources or
operations for your AWS account. The following are the service quotas for this service.

## Amazon Q Developer Pro service quotas

The features included with Amazon Q Pro subscriptions are subject to
the usage quotas outlined on the [Amazon Q Developer\
pricing page](https://aws.amazon.com/q/developer/pricing) and in the following tables. These quotas ensure fair usage and optimal
performance for all subscribers.

For more information about pricing, see the [Amazon Q Developer pricing page](https://aws.amazon.com/q/developer/pricing).
