# Favorites in the AWS Management Console

To access your frequently used services and applications more quickly, you can save their service
consoles to a list of **Favorites**. You can add and remove favorites using the AWS Management Console. When you add a service or application to your **Favorites**, it appears on the Favorites quickbar.
