# Deleting users

As an AWS Supply Chain administrator, you can delete users from the
AWS Supply Chain web application. Follow these steps to delete users.

1. On the AWS Supply Chain dashboard, from the left navigation pane, choose
    the **Settings** icon.

2. Choose **Permissions**, and then choose
    **Users**.

The **Manage Users** page appears.

3. On the **Manage Users** page, select the user that you
    want to delete and choose the **Delete** icon.
