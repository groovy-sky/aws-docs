# AWS Toolkit for Visual Studio Documentation

The AWS Toolkit for Visual Studio is a plug-in for the Visual Studio integrated development
environment (IDE). The Toolkit for Visual Studio makes it easier for developers to develop, debug, and
deploy .NET applications that use Amazon Web Services.
