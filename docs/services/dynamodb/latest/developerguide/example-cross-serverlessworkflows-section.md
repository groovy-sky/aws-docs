# Use Step Functions to invoke Lambda functions

The following code example shows how to create an AWS Step Functions state machine that invokes AWS Lambda functions in sequence.

Java

**SDK for Java 2.x**

Shows how to create an AWS serverless workflow by using AWS Step Functions and the AWS SDK for Java 2.x.
Each workflow step is implemented using an AWS Lambda function.

For complete source code and instructions on how to set up and run, see the full example on
[GitHub](https://github.com/awsdocs/aws-doc-sdk-examples/tree/main/javav2/usecases/creating_workflows_stepfunctions).

###### Services used in this example

- DynamoDB

- Lambda

- Amazon SES

- Step Functions
