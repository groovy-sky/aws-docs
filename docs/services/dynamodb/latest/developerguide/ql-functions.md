# Use PartiQL functions with DynamoDB

PartiQL in Amazon DynamoDB supports the following built-in variants of SQL standard
functions.

###### Note

Any SQL functions that are not included in this list are not currently supported in
DynamoDB.

## Aggregate functions

- [Using the SIZE function with PartiQL for amazon DynamoDB](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/ql-functions.size.html)

## Conditional functions

- [Using the EXISTS function with PartiQL for DynamoDB](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/ql-functions.exists.html)

- [Using the ATTRIBUTE\_TYPE function with PartiQL for DynamoDB](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/ql-functions.attribute_type.html)

- [Using the BEGINS\_WITH function with PartiQL for DynamoDB](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/ql-functions.beginswith.html)

- [Using the CONTAINS function with PartiQL for DynamoDB](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/ql-functions.contains.html)

- [Using the MISSING function with PartiQL for DynamoDB](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/ql-functions.missing.html)
