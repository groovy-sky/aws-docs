# Amazon Linux 2 09/25/2018 release notes

These are release notes for Amazon Linux 2.

## Major updates

Amazon Linux 2 now supports 32-bit libraries and compatibility packages. With this update, you can run 32-bit applications on Amazon Linux 2. You can run
`yum update` on an Amazon Linux 2 instance to get the full 32-bit support. This release also contains bug fixes and updates for several
Amazon Linux 2 packages from their upstream projects.
