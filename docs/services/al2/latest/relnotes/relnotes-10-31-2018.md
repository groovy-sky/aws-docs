# Amazon Linux 2 10/31/2018 release notes

These are release notes for Amazon Linux 2.

## Major updates

The OpenSSH daemon configuration file /etc/ssh/sshd\_config has been updated. The
AuthorizedKeysCommand value is configured to point to a customized script,
/opt/aws/bin/curl\_authorized\_keys to support an upcoming feature to read SSH public keys;
from the EC2 instance metadata during the SSH connection process.
