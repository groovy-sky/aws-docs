# AWS MCP Server Documentation

AWS MCP Server is a fully managed remote Model Context Protocol (MCP) server that provides AI assistants and agents with secure, authenticated access to AWS services
through natural language interactions. You can use AWS MCP Server to perform complex, multi-step AWS tasks by combining real-time access to AWS documentation, syntactically
correct API calls, and pre-built workflows called Agent SOPs that follow AWS best practices.
