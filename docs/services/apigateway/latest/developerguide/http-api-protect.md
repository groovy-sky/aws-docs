# Protect your HTTP APIs in API Gateway

API Gateway provides a number of ways to protect your API from certain threats, like malicious
users or spikes in traffic. You can protect your API using strategies like setting
throttling targets, and enabling mutual TLS. In this section you can learn how to enable
these capabilities using API Gateway.
