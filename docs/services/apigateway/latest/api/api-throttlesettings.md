# ThrottleSettings

The API request rate limits.

## Contents

**burstLimit**

The API target request burst rate limit. This allows more requests through for a period of time than the target rate limit.

Type: Integer

Required: No

**rateLimit**

The API target request rate limit.

Type: Double

Required: No
