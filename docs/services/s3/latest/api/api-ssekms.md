# SSEKMS

Specifies the use of SSE-KMS to encrypt delivered inventory reports.

## Contents

**KeyId**

Specifies the ID of the AWS Key Management Service (AWS KMS) symmetric encryption customer managed key to use for encrypting
inventory reports.

Type: String

Required: Yes
