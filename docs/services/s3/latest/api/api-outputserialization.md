# OutputSerialization

Describes how results of the Select job are serialized.

## Contents

**CSV**

Describes the serialization of CSV-encoded Select results.

Type: [CSVOutput](https://docs.aws.amazon.com/AmazonS3/latest/API/API_CSVOutput.html) data type

Required: No

**JSON**

Specifies JSON as request's output serialization format.

Type: [JSONOutput](https://docs.aws.amazon.com/AmazonS3/latest/API/API_JSONOutput.html) data type

Required: No
