# Stats

Container for the stats details.

## Contents

**BytesProcessed**

The total number of uncompressed object bytes processed.

Type: Long

Required: No

**BytesReturned**

The total number of bytes of records payload data returned.

Type: Long

Required: No

**BytesScanned**

The total number of object bytes scanned.

Type: Long

Required: No
