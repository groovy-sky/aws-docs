# S3ObjectLockLegalHold

Whether S3 Object Lock legal hold will be applied to objects in an S3 Batch Operations
job.

## Contents

**Status**

The Object Lock legal hold status to be applied to all objects in the Batch Operations
job.

Type: String

Valid Values: `OFF | ON`

Required: Yes
