# S3 API Reference

This section contains the Amazon S3 API Reference documentation, which includes
_actions_ (operations) and data types.

The S3 API reference groups each of its [Actions](api-operations.md) and [Data Types](api-types.md) into three sets: _Amazon S3_,
_Amazon S3 Control_, and _Amazon S3 on Outposts_. There
is no functional distinction between the three sets. If you don't find an API operation
or data type that you're looking for in one set, check one of the other sets.

_Actions_

- [Amazon S3](api-operations-amazon-simple-storage-service.md) — API
operations that apply bucket-level and object-level actions.

- [Amazon S3 Control](api-operations-aws-s3-control.md) — API operations for
managing all other S3 resources.

- [Amazon S3 on Outposts](api-operations-amazon-s3-on-outposts.md) — API
operations for use with Amazon S3 on Outposts. You communicate with your Outposts
bucket using an access point and endpoint connection over a virtual private
cloud (VPC).

- [Amazon S3 Tables](api-operations-amazon-s3-tables.md) — API operations for use with Amazon S3 Tables.

- [Amazon S3 Vectors](api-operations-amazon-s3-vectors.md) — API operations for use with Amazon S3 Vectors.

- [Amazon S3 Files](api-operations-amazon-s3-files.md) — API operations for use with S3 Files.

_Data types_

- [Amazon S3](api-types-amazon-simple-storage-service.md) — Data types
of API operations that apply bucket-level and object-level actions.

- [Amazon S3 Control](api-types-aws-s3-control.md) — Data types of API
operations for managing all other S3 resources.

- [Amazon S3 on Outposts](api-types-amazon-s3-on-outposts.md) — Data types of API
operations for use with Amazon S3 on Outposts.

- [Amazon S3 Tables](api-types-amazon-s3-tables.md) — Data types of API
operations for use with Amazon S3 Tables.

- [Amazon S3 Vectors](api-types-amazon-s3-vectors.md) — Data types of API
operations for use with Amazon S3 Vectors.

- [Amazon S3 Files](api-types-amazon-s3-files.md) — Data types of API
operations for use with S3 Files.
