# InventoryEncryption

Contains the type of server-side encryption used to encrypt the S3 Inventory results.

## Contents

**SSEKMS**

Specifies the use of SSE-KMS to encrypt delivered inventory reports.

Type: [SSEKMS](https://docs.aws.amazon.com/AmazonS3/latest/API/API_SSEKMS.html) data type

Required: No

**SSES3**

Specifies the use of SSE-S3 to encrypt delivered inventory reports.

Type: [SSES3](https://docs.aws.amazon.com/AmazonS3/latest/API/API_SSES3.html) data type

Required: No
