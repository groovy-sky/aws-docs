# AbortIncompleteMultipartUpload

The container for abort incomplete multipart upload

## Contents

**DaysAfterInitiation**

Specifies the number of days after which Amazon S3 aborts an incomplete multipart upload to
the Outposts bucket.

Type: Integer

Required: No
