# AbortIncompleteMultipartUpload

Specifies the days since the initiation of an incomplete multipart upload that Amazon S3 will wait before
permanently removing all parts of the upload. For more information, see [Aborting\
Incomplete Multipart Uploads Using a Bucket Lifecycle Configuration](../dev/mpuoverview.md#mpu-abort-incomplete-mpu-lifecycle-config) in the
_Amazon S3 User Guide_.

## Contents

**DaysAfterInitiation**

Specifies the number of days after which Amazon S3 aborts an incomplete multipart upload.

Type: Integer

Required: No
