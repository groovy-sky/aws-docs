# Set an instance maintenance policy on your Auto Scaling group

You can create an instance maintenance policy when you create an Auto Scaling group. You can
also create it for existing groups.

By setting an instance maintenance policy on your Auto Scaling group, you no longer have to
specify values for minimum and maximum healthy percentage parameters for the instance
refresh feature unless you want to override the instance maintenance policy.

In the console, Amazon EC2 Auto Scaling provides options to help you get started.
