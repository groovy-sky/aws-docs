# Announcement: App Runner sets end of support for specific runtime versions in December 2025

**Announcement date:** August 28, 2025

Following the End of Life (EOL) declarations by their official providers or community,
AWS App Runner will set the status of the following managed language runtime versions to End of
Support:

End of Support on **December 1, 2025**:

- Node.js: 12, 14, 16, 18

- Python: 3.7, 3.8

- .NET: 6

- Go: 1.18

- Ruby: 3.1

End of Support on **December 31, 2025**:

- PHP: 8.1

For recommendations and more information, see [End of support for managed runtime versions](../dg/service-source-code.md#service-source-code.managed-platforms.eos) in the
_AWS App Runner Developer Guide_ guide.
