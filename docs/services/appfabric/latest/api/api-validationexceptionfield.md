# ValidationExceptionField

The input failed to meet the constraints specified by the AWS service in
a specified field.

## Contents

**message**

A message about the validation exception.

Type: String

Required: Yes

**name**

The field name where the invalid entry was detected.

Type: String

Required: Yes
