# AppBundleSummary

Contains a summary of an app bundle.

## Contents

**arn**

The Amazon Resource Name (ARN) of the app bundle.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 1011.

Pattern: `arn:.+`

Required: Yes
