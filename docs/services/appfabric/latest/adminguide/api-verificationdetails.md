# VerificationDetails

The AWS AppFabric for productivity feature is in preview and is subject to change.

Contains the status and reason for the AppClient verification.

###### Topics

ParameterDescription

**verificationStatus**

The AppClient verification status.

Type: String

Valid Values: `pending_verification | verified |
                                        rejected`

Required: Yes

**statusReason**

The AppClient verification status reason.

Type: String

Length Constraints: Minimum length of 1. Maximum length of
1024.

Required: No
