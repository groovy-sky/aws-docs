# Using Amazon EventBridge

You can use [Amazon EventBridge](../../../eventbridge/latest/userguide.md) (formerly CloudWatch Events) to automate
your AWS services and respond automatically to system events such as application
availability issues or resource changes. Events from AWS services, including ACM,
are delivered to Amazon EventBridge in near-real time. You can use events to trigger targets
including AWS Lambda functions, AWS Batch jobs, Amazon SNS topics, and many others. For more
information, see [What Is\
Amazon EventBridge?](../../../eventbridge/latest/userguide/what-is-amazon-eventbridge.md)
