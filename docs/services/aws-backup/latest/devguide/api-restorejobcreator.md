# RestoreJobCreator

Contains information about the restore testing plan that
AWS Backup used to initiate the restore job.

## Contents

**RestoreTestingPlanArn**

An Amazon Resource Name (ARN) that uniquely identifies
a restore testing plan.

Type: String

Required: No
