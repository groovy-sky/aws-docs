# ScanResultInfo

Contains information about the results of a scan job.

## Contents

**ScanResultStatus**

The status of the scan results.

Valid values: `THREATS_FOUND` \| `NO_THREATS_FOUND`.

Type: String

Valid Values: `NO_THREATS_FOUND | THREATS_FOUND`

Required: Yes
