# ExpressLink Documentation

ExpressLink hardware modules are pre-programmed to connect to AWS IoT services and are pre-loaded
with security credentials. They allow original equipment manufacturers (OEMs) to connect many different products to the
cloud, send data and commands using simple commands, and use AWS services to address their data storage,
analytics, and machine learning (ML) needs.
