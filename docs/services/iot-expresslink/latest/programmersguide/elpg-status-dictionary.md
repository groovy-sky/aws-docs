# 7 Status dictionary

The status dictionary has been removed since rev 0.6 of the ExpressLink Specifications.
See [4.7.1 CONNECT?   »Request the connection status«](elpg-commands.md#elpg-connectq-command) for the current connection status.
See [9.2 OTA commands](https://docs.aws.amazon.com/iot-expresslink/latest/programmersguide/elpg-ota-updates.html#elpg-ota-commands) for the current OTA process status.
The Overflow mechanism does not have a corresponding status bit associated anymore.

## 7.1 State commands

The _STAT_ command is not available since version 0.6
