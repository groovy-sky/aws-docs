# Enable Internet Access for Your App Block Builder in Amazon WorkSpaces Applications

If you plan to enable internet access for your app block builder, you
must do so when you create the app block builder.

###### To enable internet access for an app block builder

1. Complete the steps in [Create an App Block Builder](create-app-block-builder.md) up to **Step 2: Configure Network**.

2. Choose the VPC with a NAT gateway.

3. If **Subnet** is empty, select a subnet.

4. Continue with the steps in [Create an App Block Builder](create-app-block-builder.md).
