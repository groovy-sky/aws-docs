# Configure a Connection Method for Your WorkSpaces Applications Client Users

After you install the WorkSpaces Applications client on your users' local computers, they can use
the WorkSpaces Applications client to connect to a streaming session. Depending on your
organizational requirements, you can provide client users with access to WorkSpaces Applications by doing one of the following: Setting up identity federation using SAML 2.0, using an WorkSpaces Applications user pool, or creating a streaming URL.
