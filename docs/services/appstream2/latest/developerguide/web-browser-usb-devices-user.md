# USB Devices

USB devices are not supported for browser-based WorkSpaces Applications streaming sessions. To use your USB devices with applications streamed through WorkSpaces Applications, you must use the WorkSpaces Applications client. For more information, see [WorkSpaces Applications Client Application for Windows](client-application-windows-user.md).
