# Configure a Connection Method for Your WorkSpaces Applications Users (Web Browser)

Depending on your
organizational requirements, you can provide users with access to WorkSpaces Applications through a web browser by doing one of the following: Setting up identity federation using SAML 2.0, using an WorkSpaces Applications user pool, or creating a streaming URL.
