# Prerequisites for Local Printer Redirection

To ensure that your users can use local printer redirection, you must:

- Use an image that uses a version of the WorkSpaces Applications agent released on or
after July 30, 2020. For more information, see [WorkSpaces Applications Agent Release Notes](agent-software-versions.md).

- Ensure that your users have WorkSpaces Applications client version 1.1.179 or later
installed. For more information, see [WorkSpaces Applications Windows Client Release Notes](client-release-versions.md).

- Ensure printer redirection is enabled on the stack that your users
access for streaming sessions.
