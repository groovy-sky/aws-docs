# Create and Run Athena Queries

To query your usage reports by using Athena, perform the following steps.

###### Note

Charges may apply to Athena queries that you run. For more information, see [Amazon Athena Pricing](https://aws.amazon.com/athena/pricing).

1. Open the Athena console at
    [https://console.aws.amazon.com/athena/](https://console.aws.amazon.com/athena/home).

2. Under **Database**, choose **appstream-usage**.

3. In the query pane, enter a SQL query, and choose **Run query**.
