# File Storage Options

If your WorkSpaces Applications administrator has enabled it, you can use one or more of the following storage options for your files and folders during application streaming sessions.

- [Home folders](home-folders-end-user.md)

- [Google Drive](google-drive-end-user.md)

- [OneDrive\
for Business](onedrive-end-user.md)

- [Custom shared folders](custom-shared-folders-end-user.md)

###### Note

Google Drive and OneDrive for Business are currently not supported for
Linux-based streaming instances.
