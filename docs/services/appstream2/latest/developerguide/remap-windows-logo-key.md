# Remap the Windows Logo Key or Command Key

You can remap the Mac Option and Command keys on your keyboard.

A _modifier_ key modifies the action of another key when
you use both keys together. You can use a modifier key with another key to
perform a task such as printing. A _Meta_ key is a special
type of modifier key. You can use a Meta key to temporarily change the function
of another key when you use both keys together.

To remap the Mac Option and Command keys, choose **Settings**
and **Keyboard & Mouse**.

You can remap the Option key to the following keys during a streaming
session:

- Remote Alt key

- Local modifier key

You can remap the Command key to the following keys during a streaming
session:

- Remote Control key

- Meta key
