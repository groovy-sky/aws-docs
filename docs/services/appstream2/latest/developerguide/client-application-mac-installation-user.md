# Setup and installation for macOS

Follow these steps to download and install the client application.

1. On your macOS device, open [Amazon WorkSpaces Applications\
    Downloads](https://clients.amazonappstream.com/), and choose the macOS link.

2. Download and install the application.

3. Verify that the WorkSpaces Applications client application icon appears on your Mac
    Launchpad, or check in
    `/Users/username/Applications/` or
    `~/Applications`.
