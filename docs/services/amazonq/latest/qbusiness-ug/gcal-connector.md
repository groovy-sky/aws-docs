# Connecting Google Calendar to Amazon Q Business (Preview)

###### Note

The Google Calendar connector is in preview release and is subject to change.

Google Calendar is an online calendar tool developed by Google. You can
connect a Google Calendar instance to Amazon Q Business—using either the
AWS Management Console or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
