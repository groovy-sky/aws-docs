# Connecting Amazon FSx (Windows) to Amazon Q Business

Amazon FSx (Windows) is a fully managed, cloud based file server system that offers shared
storage capabilities. You can connect your Amazon FSx (Windows) instance to Amazon Q Business—using either the AWS Management Console, CLI, or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web
experience.

The Amazon Q Amazon FSx (Windows) data source connector supports only Amazon FSx for Windows.
