# Known limitations for the Slack connector

The Slack connector has the following known limitations:

- Due to API limitations, the Amazon Q Slack connector
can only retrieve a maximum of 100 pages, with 100 files per page. Given this
limitation, the Slack connector can only crawl a maximum of 10,000
files per channel.

[Document Conventions](https://docs.aws.amazon.com/general/latest/gr/docconventions.html)
