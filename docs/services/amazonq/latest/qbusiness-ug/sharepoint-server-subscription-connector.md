# Connecting SharePoint Server (Subscription Edition) to Amazon Q Business

Microsoft SharePoint is a collaborative website building service that lets
you customize web content and create web pages, web sites, document libraries, and lists.
You can connect a SharePoint Server (Subscription Edition) instance to Amazon Q Business—using either
the AWS Management Console or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web
experience.

Amazon Q supports Microsoft SharePoint Server (2016, 2019, and
Subscription Edition).
