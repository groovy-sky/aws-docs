# Connecting Jira to Amazon Q Business

Jira is a project management tool for software development, product
management, and bug tracking. You can connect your Jira instance to Amazon Q Business—using either the AWS Management Console, CLI, or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
