# Connecting Confluence (Cloud) to Amazon Q Business

Atlassian Confluence is a collaborative work-management tool designed for
sharing, storing, and working on project planning, software development, and product
management. You can connect Confluence (Cloud) instance to Amazon Q Business—using either the AWS Management Console or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
