# Connecting Asana to Amazon Q Business (Preview)

###### Note

The Asana connector is in preview release and is subject to change.

Asana is a tool for organizing and assigning tasks among users. You can
connect a Asana instance to Amazon Q Business—using either the
AWS Management Console or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
