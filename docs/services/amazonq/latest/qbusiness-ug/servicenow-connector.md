# Connecting ServiceNow Online to Amazon Q Business

ServiceNow provides a cloud-based service management system to create and
manage organization-level workflows, such as IT services, ticketing systems, and support.
You can connect ServiceNow Online instance to Amazon Q Business—using either
the AWS Management Console or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
