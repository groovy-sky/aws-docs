# Connecting GitHub (Cloud) to Amazon Q Business

GitHub (Cloud) is a web-based hosting service for software development providing code
storage and management services with version control. You can connect your
GitHub (Cloud) instance to Amazon Q Business—using either the AWS Management Console,
CLI, or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
