# Connecting Slack to Amazon Q Business

Slack is an enterprise communications app that lets users send messages and
attachments through various public and private channels. You can connect your
Slack instance to Amazon Q Business—using either the AWS Management Console,
CLI, or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web experience.
