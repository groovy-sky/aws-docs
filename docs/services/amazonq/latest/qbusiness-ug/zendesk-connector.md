# Connecting Zendesk to Amazon Q Business

Zendesk is a customer relationship management system that helps businesses
automate and enhance customer support interactions. You can connect a Zendesk
instance to Amazon Q Business—using either the AWS Management Console or the [CreateDataSource](../api-reference/api-createdatasource.md) API—and create an Amazon Q web
experience.
