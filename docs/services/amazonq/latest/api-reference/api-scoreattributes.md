# ScoreAttributes

Provides information about the relevance score of content.

## Contents

**scoreConfidence**

The confidence level of the relevance score.

Type: String

Valid Values: `VERY_HIGH | HIGH | MEDIUM | LOW | NOT_AVAILABLE`

Required: No
