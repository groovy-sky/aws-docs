# QAppsConfiguration

Configuration information about Amazon Q Apps.

## Contents

**qAppsControlMode**

Status information about whether end users can create and use Amazon Q Apps in the web
experience.

Type: String

Valid Values: `ENABLED | DISABLED`

Required: Yes
