# PermissionOutput

The permission granted to the Amazon Q App.

## Contents

**action**

The action associated with the permission.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 20.

Valid Values: `read | write`

Required: Yes

**principal**

The principal user to which the permission applies.

Type: [PrincipalOutput](https://docs.aws.amazon.com/amazonq/latest/api-reference/API_qapps_PrincipalOutput.html) object

Required: Yes
