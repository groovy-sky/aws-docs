# DeleteDocument

A document deleted from an Amazon Q Business data source connector.

## Contents

**documentId**

The identifier of the deleted document.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 1825.

Pattern: `\P{C}*`

Required: Yes
