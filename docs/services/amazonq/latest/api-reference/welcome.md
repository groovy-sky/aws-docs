# Welcome

## QBusiness

This is the _Amazon Q Business_ API Reference. Amazon Q Business is a fully
managed, generative-AI powered enterprise chat assistant that you can deploy within your
organization. Amazon Q Business enhances employee productivity by supporting key tasks such
as question-answering, knowledge discovery, writing email messages, summarizing text,
drafting document outlines, and brainstorming ideas. Users ask questions of
Amazon Q Business and get answers that are presented in a conversational manner. For an
introduction to the service, see the [_Amazon Q Business User Guide_](../business-use-dg/what-is.md).

For an overview of the Amazon Q Business APIs, see [Overview of Amazon Q Business API operations](../business-use-dg/api-ref.md#api-overview).

For information about the IAM access control permissions you need to
use this API, see [IAM roles for Amazon Q Business](../business-use-dg/iam-roles.md) in the
_Amazon Q Business User Guide_.

## QApps

The Amazon Q Apps feature capability within Amazon Q Business allows web experience users to create
lightweight, purpose-built AI apps to fulfill specific tasks from within their web experience.
For example, users can create a Q App that exclusively generates marketing-related content to
improve your marketing team's productivity or a Q App for writing customer emails and
creating promotional content using a certain style of voice, tone, and branding. For more
information on the capabilities, see [Amazon Q Apps capabilities](../qbusiness-ug/deploy-experience-iam-role.md#q-apps-actions) in the _Amazon Q Business User_
_Guide_.

For an overview of the Amazon Q App APIs, see [Overview of Amazon Q Apps API\
operations](api-operations-qapps.md).

For information about the IAM access control permissions you need to use
the Amazon Q Apps API, see [IAM role for the Amazon Q Business web experience including Amazon Q Apps](../qbusiness-ug/deploy-experience-iam-role.md) in
the _Amazon Q Business User Guide_.
