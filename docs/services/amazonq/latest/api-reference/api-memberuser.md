# MemberUser

The users that belong to a group.

## Contents

**userId**

The identifier of the user you want to map to a group.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 1024.

Pattern: `\P{C}*`

Required: Yes

**type**

The type of the user.

Type: String

Valid Values: `INDEX | DATASOURCE`

Required: No
