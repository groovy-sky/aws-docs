# HallucinationReductionConfiguration

Configuration information required to setup hallucination reduction. For more
information, see [hallucination\
reduction](../qbusiness-ug/hallucination-reduction.md).

###### Note

The hallucination reduction feature won't work if chat orchestration controls are
enabled for your application.

## Contents

**hallucinationReductionControl**

Controls whether hallucination reduction has been enabled or disabled for your
application. The default status is `DISABLED`.

Type: String

Valid Values: `ENABLED | DISABLED`

Required: No
