# MediaExtractionConfiguration

The configuration for extracting information from media in documents.

## Contents

**audioExtractionConfiguration**

Configuration settings for extracting and processing audio content from media files.

Type: [AudioExtractionConfiguration](https://docs.aws.amazon.com/amazonq/latest/api-reference/API_AudioExtractionConfiguration.html) object

Required: No

**imageExtractionConfiguration**

The configuration for extracting semantic meaning from images in documents.
For more information, see [Extracting semantic meaning from images and visuals](../qbusiness-ug/extracting-meaning-from-images.md).

Type: [ImageExtractionConfiguration](https://docs.aws.amazon.com/amazonq/latest/api-reference/API_ImageExtractionConfiguration.html) object

Required: No

**videoExtractionConfiguration**

Configuration settings for extracting and processing video content from media files.

Type: [VideoExtractionConfiguration](https://docs.aws.amazon.com/amazonq/latest/api-reference/API_VideoExtractionConfiguration.html) object

Required: No
