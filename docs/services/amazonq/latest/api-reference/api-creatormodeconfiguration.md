# CreatorModeConfiguration

Configuration information required to invoke chat in `CREATOR_MODE`.

For more information, see [Admin controls and guardrails](../qbusiness-ug/guardrails.md) and [Conversation settings](../business-use-dg/using-web-experience.md#chat-source-scope).

## Contents

**creatorModeControl**

Status information about whether `CREATOR_MODE` has been enabled or
disabled. The default status is `DISABLED`.

Type: String

Valid Values: `ENABLED | DISABLED`

Required: Yes
