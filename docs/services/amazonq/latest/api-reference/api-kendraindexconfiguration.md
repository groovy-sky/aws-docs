# KendraIndexConfiguration

Stores an Amazon Kendra index as a retriever.

## Contents

**indexId**

The identifier of the Amazon Kendra index.

Type: String

Length Constraints: Fixed length of 36.

Pattern: `[a-zA-Z0-9][a-zA-Z0-9-]{35}`

Required: Yes
