# AuthChallengeRequestEvent

An authentication verification event activated by an end user request to use a custom
plugin.

## Contents

**authorizationUrl**

The URL sent by Amazon Q Business to a third party authentication server in response to an
authentication verification event activated by an end user request to use a custom
plugin.

Type: String

Length Constraints: Minimum length of 1. Maximum length of 2048.

Pattern: `(https?|ftp|file)://([^\s]*)`

Required: Yes
