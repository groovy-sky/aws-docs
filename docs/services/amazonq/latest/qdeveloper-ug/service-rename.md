# Amazon Q Developer rename - Summary of changes

On April 30, 2024, Amazon CodeWhisperer became a part of Amazon Q Developer. This section points you to
the parts of this guide where you can find documentation for features that you are accustomed
to using through CodeWhisperer.

As you transition from using CodeWhisperer to using Amazon Q Developer, you may consider the following changes to be most significant:

- The [administrative setup](getting-started-q-dev.md) at the professional tier (Amazon Q Developer Pro) is different than it was for CodeWhisperer Professional.

- You can [chat with Amazon Q Developer](q-on-aws.md) in the AWS Management Console, and on the AWS documentation and marketing websites.

The following familiar features of CodeWhisperer are available as part of Amazon Q Developer, with a few changes:

- Coding suggestions [in a third-party IDE](q-in-ide-setup.md)

- Coding suggestions [in the context of another AWS service](setting-up-aws-coding-env.md)

- Suggestions [at the command line](command-line.md)

- [Code reviews](code-reviews.md)

- [Dashboard](dashboard.md)
