# Amazon Q Documentation

Amazon Q Business is a generative artificial intelligence (generative AI)-powered
assistant that you can tailor to your business needs. Amazon Q Developer is an AWS generative AI assistant that helps you understand, build, extend, and operate applications
and workloads on AWS.
