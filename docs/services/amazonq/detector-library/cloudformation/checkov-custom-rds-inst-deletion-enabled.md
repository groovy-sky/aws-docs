# RDS Instance Deletion Protection [High](https://docs.aws.amazon.com/amazonq/detector-library/cloudformation/severity/high)

RDS instances with DeletionProtection property is not present or set to false. Make sure that RDS instances with DeletionProtection property sets to true.

**Detector ID**

cloudformation/checkov-custom-rds-inst-deletion-enabled@v1.0

**Category**

[Security](https://docs.aws.amazon.com/amazonq/detector-library/cloudformation/categories/security)

**Common Weakness Enumeration (CWE)![external icon](https://docs.aws.amazon.com/amazonq/detector-library/_next/static/media/external.cdeb5c57.svg)**

-

**Tags**

[\# aws-cloudformation](https://docs.aws.amazon.com/amazonq/detector-library/cloudformation/tags/aws-cloudformation)

```yaml
1Resources:
2      ExampleRDS:
3        Type: "AWS::RDS::DBInstance"

5          DBInstanceClass: !Ref DBInstanceType

8          KmsKeyId: !Ref MyKey
9          MonitoringInterval: 60
10          MultiAZ: true
11          StorageEncrypted: true
12          # Noncompliant: `DeletionProtection` is set to false
```
