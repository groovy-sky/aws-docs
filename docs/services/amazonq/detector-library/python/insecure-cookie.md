# Insecure cookie [High](https://docs.aws.amazon.com/amazonq/detector-library/python/severity/high)

Insecure cookie settings can lead to unencrypted cookie transmission. Even if a cookie doesn't contain sensitive data now, it could be added later. It's good practice to transmit all cookies only through secure channels.

**Detector ID**

python/insecure-cookie@v1.0

**Category**

[Security](https://docs.aws.amazon.com/amazonq/detector-library/python/categories/security)

**Common Weakness Enumeration (CWE)![external icon](https://docs.aws.amazon.com/amazonq/detector-library/_next/static/media/external.cdeb5c57.svg)**

[CWE-614](https://cwe.mitre.org/data/definitions/614.html) [CWE-311](https://cwe.mitre.org/data/definitions/311.html) [CWE-312](https://cwe.mitre.org/data/definitions/312.html)

**Tags**

[\# cookies](https://docs.aws.amazon.com/amazonq/detector-library/python/tags/cookies) [\# cryptography](https://docs.aws.amazon.com/amazonq/detector-library/python/tags/cryptography) [\# owasp-top10](https://docs.aws.amazon.com/amazonq/detector-library/python/tags/owasp-top10)

#### Noncompliant example

```python
1def secure_cookie_noncompliant():
2    from http.cookies import SimpleCookie
3    cookie = SimpleCookie()
4    cookie['sample'] = "sample_value"
5    # Noncompliant: the cookie is insecure.
6    cookie['sample']['secure'] = 0
7    print(cookie)
```

#### Compliant example

```python
1def secure_cookie_compliant():
2    from http.cookies import SimpleCookie
3    cookie = SimpleCookie()
4    cookie['sample'] = "sample_value"
5    # Compliant: the cookie is secure.
6    cookie['sample']['secure'] = True  # compliant
7    print(cookie)
```
