# Insecure Socket Bind [Critical](https://docs.aws.amazon.com/amazonq/detector-library/python/severity/critical)

Binding the socket with an empty IP address will allow it to accept connections from any IPv4 address provided, thus can introduce security risks.

**Detector ID**

python/insecure-socket-bind@v1.0

**Category**

[Security](https://docs.aws.amazon.com/amazonq/detector-library/python/categories/security)

**Common Weakness Enumeration (CWE)![external icon](https://docs.aws.amazon.com/amazonq/detector-library/_next/static/media/external.cdeb5c57.svg)**

[CWE-200](https://cwe.mitre.org/data/definitions/200.html)

**Tags**

[\# information-leak](https://docs.aws.amazon.com/amazonq/detector-library/python/tags/information-leak) [\# owasp-top10](https://docs.aws.amazon.com/amazonq/detector-library/python/tags/owasp-top10) [\# top25-cwes](https://docs.aws.amazon.com/amazonq/detector-library/python/tags/top25-cwes)

#### Noncompliant example

```python
1def insecure_socket_bind_noncompliant():
2    import socket
3    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
4    # Noncompliant: Empty IP Address is passed when binding to a socket.
5    s.bind(('', 0))
```

#### Compliant example

```python
1def insecure_socket_bind_compliant():
2    import socket
3    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
4    # Compliant: Non-empty IP Address is passed when binding to a socket.
5    s.bind(('192.168.1.1', 0))
```
