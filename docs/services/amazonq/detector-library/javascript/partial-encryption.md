# Sensitive data stored unencrypted due to partial encryption [Medium](https://docs.aws.amazon.com/amazonq/detector-library/javascript/severity/medium)

Encryption that is dependent on conditional logic, such as an `if...then` clause, might cause unencrypted sensitive data to be stored. If data is encrypted along one branch of a conditional statement, then encrypt data along all branches.

**Detector ID**

javascript/partial-encryption@v1.0

**Category**

[Security](https://docs.aws.amazon.com/amazonq/detector-library/javascript/categories/security)

**Common Weakness Enumeration (CWE)![external icon](https://docs.aws.amazon.com/amazonq/detector-library/_next/static/media/external.cdeb5c57.svg)**

[CWE-311](https://cwe.mitre.org/data/definitions/311.html)
