### [Protection mechanism failure](https://docs.aws.amazon.com/amazonq/detector-library/jsx/protection-mechanism-failure)

### [Pseudorandom number generators](https://docs.aws.amazon.com/amazonq/detector-library/jsx/pseudorandom-number-generator)

Using pseudorandom number generators (PRNGs) is security-sensitive.

### [File injection](https://docs.aws.amazon.com/amazonq/detector-library/jsx/file-injection)

Writing unsanitized user data to a file is unsafe.
