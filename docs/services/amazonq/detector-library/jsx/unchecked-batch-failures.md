# Data loss in a batch request [Medium](https://docs.aws.amazon.com/amazonq/detector-library/jsx/severity/medium)

DynamoDB batch operations can tolerate the failure of individual requests. However, not all requested items are guaranteed to be processed which can compromise data integrity. Implement retry logic and use `UnprocessedKeys` or `UnprocessedItems` to check if all requests succeeded.
