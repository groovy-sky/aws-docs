# Aurora DSQL API reference

In addition to the AWS Management Console and the AWS Command Line Interface (AWS CLI), Aurora DSQL also provides an API interface.
You can use the API operations to manage your resources in Aurora DSQL.

For an alphabetical list of API operations, see [Actions](../../../../reference/aurora-dsql/latest/apireference/api-operations.md).

For an alphabetical list of data types, see [Data types](../../../../reference/aurora-dsql/latest/apireference/api-types.md).

For a list of common query parameters, see [Common parameters](../../../../reference/aurora-dsql/latest/apireference/commonparameters.md).

For descriptions of the error codes, see [Common errors](../../../../reference/aurora-dsql/latest/apireference/commonerrors.md).

For more information about the AWS CLI, see AWS Command Line Interface reference for Aurora DSQL.
