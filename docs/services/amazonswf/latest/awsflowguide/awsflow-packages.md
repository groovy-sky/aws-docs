# AWS Flow Framework for Java Packages

This section provides an overview of the packages included with the AWS Flow Framework for Java. For more information about
each package, see the com.amazonaws.services.simpleworkflow.flow in the [AWS SDK for Java API Reference](../../../../reference/sdk-for-java/latest/reference.md).

[com.amazonaws.services.simpleworkflow.flow](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/package-summary.md)

Contains components that integrate with Amazon SWF.

[com.amazonaws.services.simpleworkflow.flow.annotations](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/annotations/package-summary.md)

Contains the annotations used by the AWS Flow Framework for Java programming model.

[com.amazonaws.services.simpleworkflow.flow.aspectj](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/aspectj/package-summary.md)

Contains AWS Flow Framework for Java components required for features such as [@Asynchronous](annotations.md#annotations-asynchronous)
and [@ExponentialRetry](annotations.md#annotations-exponentialretry).

[com.amazonaws.services.simpleworkflow.flow.common](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/common/package-summary.md)

Contains common utilities such as framework-defined constants.

[com.amazonaws.services.simpleworkflow.flow.core](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/core/package-summary.md)

Contains core features such as `Task` and `Promise`.

[com.amazonaws.services.simpleworkflow.flow.generic](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/generic/package-summary.md)

Contains core components, such as generic clients, that other features build on.

[com.amazonaws.services.simpleworkflow.flow.interceptors](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/interceptors/package-summary.md)

Contains implementations of framework provided decorators including `RetryDecorator`.

[com.amazonaws.services.simpleworkflow.flow.junit](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/junit/package-summary.md)

Contains components that provide Junit integration.

[com.amazonaws.services.simpleworkflow.flow.pojo](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/pojo/package-summary.md)

Contains classes that implement activity and workflow definitions for the annotation-based programming model.

[com.amazonaws.services.simpleworkflow.flow.spring](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/spring/package-summary.md)

Contains components that provide Spring integration.

[com.amazonaws.services.simpleworkflow.flow.test](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/test/package-summary.md)

Contains helper classes, such as `TestWorkflowClock`, for unit testing workflow implementations.

[com.amazonaws.services.simpleworkflow.flow.worker](../../../../reference/awsjavasdk/latest/javadoc/com/amazonaws/services/simpleworkflow/flow/worker/package-summary.md)

Contains implementations of activity and workflow workers.
