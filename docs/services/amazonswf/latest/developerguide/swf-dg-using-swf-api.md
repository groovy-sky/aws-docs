# Working with Amazon SWF APIs

In addition to using the AWS SDKs that are described in [Develop with AWS SDKs](intro.md#aws-sdks), you can use the HTTP API directly.

To use the API, you send HTTP requests to the [SWF endpoint](../../../../general/latest/gr/rande.md#swf_region) that matches the region that
you want to use for your domains, workflows and activities. For more information about making HTTP requests for
Amazon SWF, see [Making HTTP Requests to Amazon SWF](usingjson-swf.md).

This section provides basic information about using the HTTP API to develop your workflows with Amazon SWF. More
advanced features, such as using timers, logging with CloudTrail and tagging your workflows are provided in the section,
[Basic workflow concepts in Amazon SWF](swf-dg-basic.md).
