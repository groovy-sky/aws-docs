# Earning additional credits

If you are a new AWS customer, you can earn an additional USD $100 in credits
regardless of your account plan by completing activities. These activities assist your
learning of AWS services. You can find the activities in the **Explore**
**AWS** widget on your [AWS Console Home\
dashboard](https://console.aws.amazon.com/).

###### Topics

- [Launch an instance using Amazon EC2](#free-tier-plans-activities-ec2)

- [Use a foundational model in the Amazon Bedrock playground](#free-tier-plans-activities-br)

- [Set up a cost budget using AWS Budgets](#free-tier-plans-activities-budgets)

- [Create a web app using AWS Lambda](#free-tier-plans-activities-lambda)

- [Create an Amazon RDS database](#free-tier-plans-activities-rds)

## Launch an instance using Amazon EC2

Use Amazon Elastic Compute Cloud (Amazon EC2) to create virtual machines, or instances, that run on the
AWS cloud. In this activity, you will learn how to quickly get started by
launching an instance in a server. You also learn how to clean up your
instances.

To learn more about Amazon EC2, see the [Amazon Elastic Compute Cloud Documentation](https://docs.aws.amazon.com/ec2).

###### Note

This activity incurs AWS service charges that are deducted from your
AWS Free Tier credits. To see your potential costs, see the service pricing page on
[Amazon EC2 On-Demand\
Pricing](https://aws.amazon.com/ec2/pricing/on-demand).

## Use a foundational model in the Amazon Bedrock playground

Use Amazon Bedrock to access a wide range of foundation model from leading AI companies.
You can generate different types of output including texts, images, videos, and
embedding output through a unified API. In this activity, you will choose a model
and submit a prompt to generate a response in the Amazon Bedrock playground.

To learn more about Amazon Bedrock, see the [Amazon Bedrock\
Documentation](https://docs.aws.amazon.com/bedrock/latest/userguide/getting-started.html).

###### Note

This activity incurs AWS service charges that are deducted from your
AWS Free Tier credits. To see your potential costs, see the service pricing page on
[Amazon Bedrock\
pricing](https://aws.amazon.com/bedrock/pricing).

## Set up a cost budget using AWS Budgets

Use AWS Budgets to set a budget that alerts you when you exceed, or are
forecasted to exceed, your budgeted cost or usage amount. In this activity, you
create a cost budget to monitor your AWS cost and usage.

To learn more about AWS Budgets, see the [AWS Budgets\
Documentation](https://docs.aws.amazon.com/cost-management/latest/userguide/budgets-managing-costs.html).

## Create a web app using AWS Lambda

Use AWS Lambda to run code without provisioning or managing servers. In this
activity, you will learn how to build a simple web app consisting of a Lambda
function with a function URL.

###### Note

This activity incurs AWS service charges that are deducted from your
AWS Free Tier credits. To see your potential costs, see the service pricing page on
[AWS Lambda\
pricing](https://aws.amazon.com/lambda/pricing).

To learn more about AWS Lambda, see [AWS Lambda Documentation](https://docs.aws.amazon.com/lambda).

## Create an Amazon RDS database

Amazon Relational Database Service (Amazon RDS) is a fully managed database service that simplify setup,
operation, and scaling of relational databases in the cloud. In this activity you
will launch an Amazon RDS or Aurora database and explore basic configuration options. You
will learn the essential settings you need to set up and manage your
database.

To learn more about Amazon RDS, see [Amazon Relational Database Service\
Documentation](https://docs.aws.amazon.com/rds).

###### Note

This activity incurs AWS service charges that are deducted from your
AWS Free Tier credits. To see your potential costs, see the service pricing page on
[Amazon RDS pricing](https://aws.amazon.com/rds/pricing).
