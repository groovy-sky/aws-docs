# Amazon AppFlow Documentation

Amazon AppFlow is a fully managed API integration service that you use to
connect your software as a service (SaaS) applications to AWS services, and
securely transfer data. Use Amazon AppFlow flows to manage and automate your data
transfers without needing to write code.
