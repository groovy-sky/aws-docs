# Getting started with RDS Proxy

Use the information in the following pages to set up and manage [Amazon RDS Proxy](rds-proxy.md) and
set related security options. The security options control who can access each proxy and how each proxy
connects to DB instances.

If you're new to RDS Proxy, we recommend following the pages in the order that we present them.
