# Migrating synchronously from Linux to Linux for Amazon RDS for Db2

With
this migration approach, you set up replication between your self-managed Db2 database
and your Amazon RDS for Db2 DB instance. Changes made to the self-managed database replicates
to the RDS for Db2 DB instance in near real-time. This approach can provide continuous
availability and minimize downtime during the migration process.
