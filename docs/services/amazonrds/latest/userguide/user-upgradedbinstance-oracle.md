# Upgrading the RDS for Oracle DB engine

When Amazon RDS supports a new version of Oracle Database, you can upgrade your DB instances to
the new version. For information about which Oracle versions are available on Amazon RDS, see
[_Amazon RDS for Oracle Release Notes_](../oraclereleasenotes/welcome.md).

###### Important

RDS for Oracle Databases 11g, 12c, and 18c are no longer supported. If you maintain Oracle Database 11g, 12c, or 18c snapshots, you can upgrade them
to a later release. For more information, see [Upgrading an Oracle DB snapshot](https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/USER_UpgradeDBSnapshot.Oracle.html).
