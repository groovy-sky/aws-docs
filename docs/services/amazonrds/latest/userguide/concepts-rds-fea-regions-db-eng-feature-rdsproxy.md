# Supported Regions and DB engines for Amazon RDS Proxy

Amazon RDS Proxy is a fully managed, highly available database proxy that makes applications more
scalable by pooling and sharing established database connections. For more information, see
[Amazon RDS Proxy](rds-proxy.md).

RDS Proxy isn't available for the following engines:

- RDS for Db2

- RDS for Oracle
