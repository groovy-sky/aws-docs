# aurora\_stat\_system\_waits

Reports wait event information for the Aurora PostgreSQL DB instance.

## Syntax

```nohighlight

aurora_stat_system_waits()
```

## Arguments

None

## Return type

SETOF record

## Usage notes

This function returns the cumulative number of waits and cumulative wait time for
each wait event generated by the DB instance that you're currently connected
to.

The returned recordset includes the following fields:

- `type_id` – The ID of the type of wait event.

- `event_id` – The ID of the wait event.

- `waits` – The number of times the wait event
occurred.

- `wait_time` – The total amount of time in microseconds
spent waiting for this event.

Statistics returned by this function are reset when a DB instance restarts.
