Contains the configuration information of the Parquet format.

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "SchemaDefinition" : SchemaDefinition
}

```

```yaml

  SchemaDefinition:
    SchemaDefinition

```
