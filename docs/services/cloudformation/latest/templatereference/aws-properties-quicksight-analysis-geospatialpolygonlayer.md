# AWS::QuickSight::Analysis GeospatialPolygonLayer

The geospatial polygon layer.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "Style" : GeospatialPolygonStyle
}

```

### YAML

```yaml

  Style:
    GeospatialPolygonStyle

```

## Properties

`Style`

The visualization style for a polygon layer.

_Required_: Yes

_Type_: [GeospatialPolygonStyle](aws-properties-quicksight-analysis-geospatialpolygonstyle.md)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
