# AWS::QuickSight::DataSet ColumnGroup

Groupings of columns that work together in certain Quick Sight features. This is
a variant type structure. For this structure to be valid, only one of the attributes can
be non-null.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "GeoSpatialColumnGroup" : GeoSpatialColumnGroup
}

```

### YAML

```yaml

  GeoSpatialColumnGroup:
    GeoSpatialColumnGroup

```

## Properties

`GeoSpatialColumnGroup`

Geospatial column group that denotes a hierarchy.

_Required_: No

_Type_: [GeoSpatialColumnGroup](https://docs.aws.amazon.com/AWSCloudFormation/latest/TemplateReference/aws-properties-quicksight-dataset-geospatialcolumngroup.html)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)

[Document Conventions](https://docs.aws.amazon.com/general/latest/gr/docconventions.html)
