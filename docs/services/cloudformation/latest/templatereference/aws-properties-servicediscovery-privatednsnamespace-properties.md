# AWS::ServiceDiscovery::PrivateDnsNamespace Properties

Properties for the
private DNS namespace.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "DnsProperties" : PrivateDnsPropertiesMutable
}

```

### YAML

```yaml

  DnsProperties:
    PrivateDnsPropertiesMutable

```

## Properties

`DnsProperties`

DNS properties for
the private DNS namespace.

_Required_: No

_Type_: [PrivateDnsPropertiesMutable](https://docs.aws.amazon.com/AWSCloudFormation/latest/TemplateReference/aws-properties-servicediscovery-privatednsnamespace-privatednspropertiesmutable.html)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)

[Document Conventions](https://docs.aws.amazon.com/general/latest/gr/docconventions.html)
