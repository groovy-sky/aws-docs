# AWS::QuickSight::Dashboard GeospatialPolygonStyle

The polygon style for a polygon layer.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "PolygonSymbolStyle" : GeospatialPolygonSymbolStyle
}

```

### YAML

```yaml

  PolygonSymbolStyle:
    GeospatialPolygonSymbolStyle

```

## Properties

`PolygonSymbolStyle`

The polygon symbol style for a polygon layer.

_Required_: No

_Type_: [GeospatialPolygonSymbolStyle](aws-properties-quicksight-dashboard-geospatialpolygonsymbolstyle.md)
