# AWS::SageMaker::EndpointConfig ExplainerConfig

A parameter to activate explainers.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "ClarifyExplainerConfig" : ClarifyExplainerConfig
}

```

### YAML

```yaml

  ClarifyExplainerConfig:
    ClarifyExplainerConfig

```

## Properties

`ClarifyExplainerConfig`

A member of `ExplainerConfig` that contains configuration parameters for
the SageMaker Clarify explainer.

_Required_: No

_Type_: [ClarifyExplainerConfig](aws-properties-sagemaker-endpointconfig-clarifyexplainerconfig.md)

_Update requires_: [Replacement](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-replacement)
