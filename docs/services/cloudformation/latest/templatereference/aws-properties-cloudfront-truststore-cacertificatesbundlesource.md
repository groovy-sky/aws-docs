# AWS::CloudFront::TrustStore CaCertificatesBundleSource

A CA certificates bundle source.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "CaCertificatesBundleS3Location" : CaCertificatesBundleS3Location
}

```

### YAML

```yaml

  CaCertificatesBundleS3Location:
    CaCertificatesBundleS3Location

```

## Properties

`CaCertificatesBundleS3Location`

The CA certificates bundle location in Amazon S3.

_Required_: Yes

_Type_: [CaCertificatesBundleS3Location](https://docs.aws.amazon.com/AWSCloudFormation/latest/TemplateReference/aws-properties-cloudfront-truststore-cacertificatesbundles3location.html)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
