# AWS::S3Outposts::AccessPoint VpcConfiguration

Contains the virtual private cloud (VPC) configuration for the specified access
point.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "VpcId" : String
}

```

### YAML

```yaml

  VpcId: String

```
