# AWS::Connect::EvaluationForm EvaluationFormBaseItem

An item at the root level. All items must be sections.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "Section" : EvaluationFormSection
}

```

### YAML

```yaml

  Section:
    EvaluationFormSection

```

## Properties

`Section`

A subsection or inner section of an item.

_Required_: Yes

_Type_: [EvaluationFormSection](https://docs.aws.amazon.com/AWSCloudFormation/latest/TemplateReference/aws-properties-connect-evaluationform-evaluationformsection.html)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
