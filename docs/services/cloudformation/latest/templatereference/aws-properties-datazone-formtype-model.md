# AWS::DataZone::FormType Model

The model of the API.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "Smithy" : String
}

```

### YAML

```yaml

  Smithy: String

```

`Smithy`

Indicates the smithy model of the API.

_Required_: No

_Type_: String

_Minimum_: `1`

_Maximum_: `100000`
