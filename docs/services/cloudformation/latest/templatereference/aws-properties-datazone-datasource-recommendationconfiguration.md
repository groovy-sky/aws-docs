# AWS::DataZone::DataSource RecommendationConfiguration

The recommendation configuration for the data source.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "EnableBusinessNameGeneration" : Boolean
}

```

### YAML

```yaml

  EnableBusinessNameGeneration: Boolean

```

## Properties

`EnableBusinessNameGeneration`

Specifies whether automatic business name generation is to be enabled or not as part of
the recommendation configuration.

_Required_: No

_Type_: Boolean

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
