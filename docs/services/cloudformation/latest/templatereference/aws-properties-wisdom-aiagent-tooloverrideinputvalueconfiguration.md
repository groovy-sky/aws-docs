# AWS::Wisdom::AIAgent ToolOverrideInputValueConfiguration

Configuration for overriding tool input values.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "Constant" : ToolOverrideConstantInputValue
}

```

### YAML

```yaml

  Constant:
    ToolOverrideConstantInputValue

```

## Properties

`Constant`

Constant input value configuration for tool override.

_Required_: Yes

_Type_: [ToolOverrideConstantInputValue](aws-properties-wisdom-aiagent-tooloverrideconstantinputvalue.md)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
