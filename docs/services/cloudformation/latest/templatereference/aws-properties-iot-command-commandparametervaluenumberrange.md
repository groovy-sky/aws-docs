# AWS::IoT::Command CommandParameterValueNumberRange

The `CommandParameterValueNumberRange` property type specifies Property description not available. for an [AWS::IoT::Command](aws-resource-iot-command.md).

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "Max" : String,
  "Min" : String
}

```

### YAML

```yaml

  Max: String
  Min: String

```
