# AWS::Transfer::Agreement CustomDirectories

The `CustomDirectories` property type specifies Property description not available. for an [AWS::Transfer::Agreement](aws-resource-transfer-agreement.md).

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "FailedFilesDirectory" : String,
  "MdnFilesDirectory" : String,
  "PayloadFilesDirectory" : String,
  "StatusFilesDirectory" : String,
  "TemporaryFilesDirectory" : String
}

```

### YAML

```yaml

  FailedFilesDirectory: String
  MdnFilesDirectory: String
  PayloadFilesDirectory: String
  StatusFilesDirectory: String
  TemporaryFilesDirectory: String

```
