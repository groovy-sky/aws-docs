# AWS::QuickSight::Dashboard DashboardSourceEntity

Dashboard source entity.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "SourceTemplate" : DashboardSourceTemplate
}

```

### YAML

```yaml

  SourceTemplate:
    DashboardSourceTemplate

```

## Properties

`SourceTemplate`

Source template.

_Required_: No

_Type_: [DashboardSourceTemplate](https://docs.aws.amazon.com/AWSCloudFormation/latest/TemplateReference/aws-properties-quicksight-dashboard-dashboardsourcetemplate.html)

_Update requires_: [No interruption](../userguide/using-cfn-updating-stacks-update-behaviors.md#update-no-interrupt)
