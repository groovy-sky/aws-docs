# AWS::Connect::Notification NotificationContent

A map of localized notification content where keys are locale codes and values are the notification text in that locale. Each localized string can be 0-500 characters.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "DeDE" : String,
  "EnUS" : String,
  "EsES" : String,
  "FrFR" : String,
  "IdID" : String,
  "ItIT" : String,
  "JaJP" : String,
  "KoKR" : String,
  "PtBR" : String,
  "ZhCN" : String,
  "ZhTW" : String
}

```

### YAML

```yaml

  DeDE: String
  EnUS: String
  EsES: String
  FrFR: String
  IdID: String
  ItIT: String
  JaJP: String
  KoKR: String
  PtBR: String
  ZhCN: String
  ZhTW: String

```
