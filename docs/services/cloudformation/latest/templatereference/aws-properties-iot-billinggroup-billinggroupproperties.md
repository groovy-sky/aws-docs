# AWS::IoT::BillingGroup BillingGroupProperties

The properties of a billing group.

## Syntax

To declare this entity in your CloudFormation template, use the following syntax:

### JSON

```json

{
  "BillingGroupDescription" : String
}

```

### YAML

```yaml

  BillingGroupDescription: String

```
