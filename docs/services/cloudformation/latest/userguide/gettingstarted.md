# Getting started with CloudFormation

You can begin using CloudFormation through the AWS Management Console by creating a stack from an example
template, which will help you learn the basics of stack creation. A
_template_ is a text file that defines all the resources within a stack.
A _stack_ is the deployment of a CloudFormation template. From a single
template, you can create multiple stacks. Each stack contains a collection of AWS resources
that can be managed as a single unit.

CloudFormation is a free service; however, you are charged for the AWS resources you include
in your stacks at the current rates for each. For more information about AWS pricing, go to
the detail page for each product on [http://aws.amazon.com](http://aws.amazon.com/).
