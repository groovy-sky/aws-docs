# Create an AWS account alias

If you want the URL for your IAM users to contain your company name (or another
easy-to-remember identifier) instead of the AWS account ID, you can create an _account alias_.

To learn how to create or update an account alias, see [Using an alias for your AWS account ID](../../../iam/latest/userguide/console-account-alias.md) in the
_IAM User Guide_.
