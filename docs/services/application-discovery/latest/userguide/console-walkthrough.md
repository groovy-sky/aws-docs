# Discovering data with the AWS Migration Hub console

AWS Application Discovery Service (Application Discovery Service) is integrated with AWS Migration Hub (Migration Hub) and customers can view and
manage their data collectors, servers, and applications within Migration Hub. When you use the
Application Discovery Service console, you are redirected to the Migration Hub console. Working with the Migration Hub console
requires no extra steps or setup on your part.

In this section, you can find how to manage and monitor Application Discovery Service Agentless Collector
(Agentless Collector) and AWS Application Discovery Agent (Discovery Agent) using the console.
