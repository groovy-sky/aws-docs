# Welcome to Amazon SimpleDB

This is the Developer Guide for _Amazon SimpleDB_. This guide provides a conceptual overview of Amazon SimpleDB, programming
reference material, and a detailed API reference.

Amazon SimpleDB is a web service for running queries on structured data in real time. This service works in close conjunction with
Amazon Simple Storage Service (Amazon S3) and Amazon Elastic Compute Cloud (Amazon EC2), collectively providing the ability
to store, process and query data sets in the cloud. These services are designed to make web-scale computing easier and more
cost-effective for developers.
