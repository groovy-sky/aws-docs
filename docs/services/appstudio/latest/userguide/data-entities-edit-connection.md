# Add or edit connected data source and map fields

###### Tip

You can press CTRL+Z to undo the most recent change to your entity.

1. If necessary, navigate to the entity you want to edit.

2. Choose the **Connection** tab to view or manage the connection between the entity and a data source table
    where data is stored when your application is published. Once a data source table is connected, you can map the entity fields to the columns
    of the table.

3. In **Connector**, choose the connector that contains a connection to the desired data source table. For more information
    about connectors, see [Connect App Studio to other services with connectors](connectors.md).

4. In **Table**, choose the table you want to use as a data source for the entity.

5. The table shows the fields of entity, and the data source column they are mapped to. Choose **Auto map** to automatically map your entity fields with your data source columns. You can also map fields manually in the table by choosing
    the data source column in the dropdown for each entity field.
