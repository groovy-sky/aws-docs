# AWS App Studio Documentation

AWS App Studio is a generative AI-powered service that uses natural language to create secure,
enterprise-grade applications fully managed by AWS. Builders can create and deploy apps to modernize business processes
such as inventory management and tracking, claims processing, and complex approvals to improve employee productivity and customer outcomes.
