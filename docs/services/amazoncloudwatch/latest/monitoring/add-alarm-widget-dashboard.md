# Adding an alarm status widget to a CloudWatch dashboard

###### To add an alarm status widget to a dashboard

1. Open the CloudWatch console at
    [https://console.aws.amazon.com/cloudwatch/](https://console.aws.amazon.com/cloudwatch).

2. In the navigation pane, choose
    **Dashboards**,
    and
    then
    choose a dashboard.

3. Choose the **+** symbol.

4. Choose **Alarm status**.

5. Select the check boxes next to the alarms that you want to add to the widget,
    and then choose **Create widget**.

6. Choose **Add to dashboard**.
