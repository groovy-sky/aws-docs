# Removing a gauge widget from a CloudWatch dashboard

###### To remove a gauge widget from a dashboard

1. Open the CloudWatch console at
    [https://console.aws.amazon.com/cloudwatch/](https://console.aws.amazon.com/cloudwatch).

2. In the navigation pane, choose **Dashboards**, and
    then
    choose the dashboard that contains the gauge widget you
    want to delete.

3. In the upper-right corner of the gauge widget
    that
    you want to delete, choose **Widget**
**actions**, and choose **Delete**.

4. Choose **Save dashboard**.
