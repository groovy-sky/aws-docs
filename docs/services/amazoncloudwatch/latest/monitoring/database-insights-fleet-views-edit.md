# Edit a fleet health view for CloudWatch Database Insights

To edit a fleet health view, use the following procedure.

###### To edit a fleet health view

1. Sign in to the AWS Management Console and open the CloudWatch console at
    [https://console.aws.amazon.com/cloudwatch/](https://console.aws.amazon.com/cloudwatch).

2. Choose **Insights**.

3. Choose **Database Insights**.

4. Choose the **Saved fleets** dropdown.

5. Choose the vertical ellipsis for a fleet health view you want to edit.

6. In the **Edit filter set (fleet)** window, you can edit the name of the fleet and the filters for the fleet.

7. Choose the **Save** button.
