# Removing a line graph widget from a CloudWatch dashboard

###### To remove a line graph widget from a dashboard

1. Open the CloudWatch console at
    [https://console.aws.amazon.com/cloudwatch/](https://console.aws.amazon.com/cloudwatch).

2. In the navigation pane, choose **Dashboards**, and
    then
    choose a dashboard.

3. In the upper-right corner of the
    line
    widget that you want to remove, choose **Widget actions**,
    and then choose **Delete**.

4. Choose **Save dashboard**.
