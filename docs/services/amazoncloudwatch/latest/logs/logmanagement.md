# Log management

CloudWatch Logs provides advanced log management capabilities that help you organize, transform, and
analyze your log data more effectively. These features include [cross-account and cross-region data\
centralization](cloudwatchlogs-centralization.md), [automatic data\
discovery and schema management,](https://docs.aws.amazon.com/AmazonCloudWatch/latest/logs/data-source-discovery-management.html) [log transformation during ingestions](https://docs.aws.amazon.com/AmazonCloudWatch/latest/logs/CloudWatch-Logs-Transformation.html),
and [enhanced analytics with facets for interactive log\
exploration](cloudwatchlogs-facets.md).
