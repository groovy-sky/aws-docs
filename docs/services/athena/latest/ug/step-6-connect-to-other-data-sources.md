# Step 6: Connect to other data sources

This tutorial used a data source in Amazon S3 in CSV format. For information about using
Athena with AWS Glue, see [Use AWS Glue Data Catalog to connect to your data](https://docs.aws.amazon.com/athena/latest/ug/data-sources-glue.html). You can also connect Athena to a variety of data
sources by using ODBC and JDBC drivers, external Hive metastores, and Athena data source
connectors. For more information, see [Connect to data sources](https://docs.aws.amazon.com/athena/latest/ug/work-with-data-stores.html).
