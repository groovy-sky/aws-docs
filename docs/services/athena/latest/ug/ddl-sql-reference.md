# SQL reference for Athena

Amazon Athena supports a subset of Data Definition Language (DDL) and Data Manipulation
Language (DML) statements, functions, operators, and data types. With some exceptions, Athena
DDL is based on [HiveQL\
DDL](https://cwiki.apache.org/confluence/display/Hive/LanguageManual+DDL) and Athena DML is based on [Trino](https://trino.io/docs/current/language.html). For information about
Athena engine versions, see [Athena engine versioning](engine-versions.md).
