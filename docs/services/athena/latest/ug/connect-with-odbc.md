# Connect to Amazon Athena with ODBC

Amazon Athena offers two ODBC drivers, versions 1.x and 2.x. The Athena ODBC 2.x driver is a
new alternative that supports Linux, macOS ARM, macOS Intel, and Windows 64-bit systems. The
Athena 2.x driver supports all authentication plugins that the 1.x ODBC driver supports, and
almost all connection parameters are backward-compatible.

- To download the ODBC 2.x driver, see [Amazon Athena ODBC 2.x](odbc-v2-driver.md).

- To download the ODBC 1.x driver, see [Athena ODBC 1.x driver](connect-with-odbc-driver-and-documentation-download-links.md).
