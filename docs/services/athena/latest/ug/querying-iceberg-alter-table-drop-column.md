# ALTER TABLE DROP COLUMN

Drops a column from an existing Iceberg table.

## Synopsis

```sql

ALTER TABLE [db_name.]table_name DROP COLUMN col_name
```

## Example

```sql

ALTER TABLE iceberg_table DROP COLUMN userid
```
