# Athena ODBC 2.x connection parameters

The **Amazon Athena ODBC Configuration** dialog box options include
**Authentication Options**, **Advanced Options**,
**Logging Options**, **Endpoint Overrides** and
**Proxy Options**. For detailed information about each, visit the
corresponding links.

- [Main ODBC 2.x connection parameters](https://docs.aws.amazon.com/athena/latest/ug/odbc-v2-driver-main-connection-parameters.html)

- [Authentication options](https://docs.aws.amazon.com/athena/latest/ug/odbc-v2-driver-authentication-options.html)

- [Advanced options](https://docs.aws.amazon.com/athena/latest/ug/odbc-v2-driver-advanced-options.html)

- [Logging options](https://docs.aws.amazon.com/athena/latest/ug/odbc-v2-driver-logging-options.html)

- [Endpoint overrides](https://docs.aws.amazon.com/athena/latest/ug/odbc-v2-driver-endpoint-overrides.html)

- [Proxy options](https://docs.aws.amazon.com/athena/latest/ug/odbc-v2-driver-proxy-options.html)
