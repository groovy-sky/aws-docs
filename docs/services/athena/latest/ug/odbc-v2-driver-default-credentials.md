# Default credentials

You can use the default credentials that you configure on your client system to
connect to Amazon Athena. For information about using default credentials, see [Using the\
Default Credential Provider Chain](../../../../reference/sdk-for-java/v1/developer-guide/credentials.md#credentials-default) in the _AWS SDK for Java Developer Guide_.
