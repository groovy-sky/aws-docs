# View recent queries in the Athena console

You can use the Athena console to see which queries succeeded or failed, and view error
details for the queries that failed. Athena keeps a query history for 45 days.

###### To view recent queries in the Athena console

1. Open the Athena console at
    [https://console.aws.amazon.com/athena/](https://console.aws.amazon.com/athena/home).

2. Choose **Recent queries**. The **Recent**
**queries** tab shows information about each query that ran.

3. To open a query statement in the query editor, choose the query's execution
    ID.

![Choose the execution ID of a query to see it in the query editor.](https://docs.aws.amazon.com/images/athena/latest/ug/images/querying-view-query-statement.png)

4. To see the details for a query that failed, choose the
    **Failed** link for the query.

![Choose the Failed link for a query to view information about the failure.](https://docs.aws.amazon.com/images/athena/latest/ug/images/querying-view-query-failure-details.png)
