# Building a backend for an application

With AWS Amplify you can build a fullstack application with data, authentication,
storage, and frontend hosting that is deployed to AWS.

AWS Amplify Gen 2 introduces a
TypeScript-based, code-first developer experience for defining backends. To learn how to use Amplify Gen 2 to build and connect a backend to
your app, see [Build &\
connect backend](https://docs.amplify.aws/nextjs/build-a-backend) in the _Amplify docs_.

If you are looking for the documentation for building a backend for a Gen 1 app, using the
CLI and Amplify Studio, see [Build & connect backend](https://docs.amplify.aws/gen1/react/build-a-backend) in the _Gen 1 Amplify docs_.
