# AWS Amplify Documentation

Use AWS Amplify to develop and deploy cloud-powered mobile and web
applications. Amplify provides frontend libraries, UI components, and backend
building for fullstack applications on AWS. Amplify Hosting
provides a continuous delivery and hosting service for fullstack cloud applications.
